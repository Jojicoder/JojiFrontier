# 反撃・反応攻撃仕様
## 状態

共通仕様確定。JOJIFrontierでは通常攻撃を受けただけでは反撃しない。反撃は効果文に
「反撃」または「反応攻撃」と明記されたスキル・武器効果だけが発生させる。

## 基本規則

- 味方・敵とも通常反撃なし
- 速度差による追撃、連続攻撃、二回攻撃なし
- 通常攻撃1回につきダメージ解決は1回
- 反撃は使用者の通常行動を消費せず、行動済みでも条件を満たせる
- 反撃回数は各効果の使用回数に従う
- 1つのRoot Actionから攻撃的反応は最大1回
- 反応攻撃に対する反撃・警戒射撃・追撃は発生しない

初期実装で反撃を行う効果は槍兵の`counterthrust`（反撃準備）だけとする。

## 反撃準備

発動条件:

1. 使用者が反撃準備を装備し、戦闘中の使用回数が残っている
2. 敵の直接単体攻撃でダメージ処理を受けた
3. 元攻撃と地形・状態処理後もHP1以上で盤面にいる
4. 攻撃者もHP1以上で盤面にいる
5. 強制移動後の両者の現在位置が装備武器の最小・最大射程内
6. 使用者が現在地から武器攻撃可能

全条件成立時だけ使用回数を消費し、攻撃者へ通常武器攻撃を1回行う。条件不成立では使用回数を
消費しない。プレイヤー確認は挟まず自動発動し、戦闘予測に「反撃準備あり」を表示する。

## 状態・地形との関係

- よろめきは移動だけを禁止するため、現在地から反撃可能
- 移動低下は反撃へ影響しない
- 毒、炎上、防御低下中でも反撃可能
- HP0、撤退、降伏、Boss専用スタン中は反撃不可
- 煙幕や茂みに回避規則がある場合、反撃にも同じ攻撃規則を適用
- 武器の最小射程を無視しない。隣接攻撃不可の武器は隣接相手へ反撃できない
- ノックバックされた場合は移動後の位置で射程を再判定する

状態異常は反撃を一律禁止しない。反応禁止を持つ将来効果を追加する場合は、独立した
`reactionsDisabled`能力として明記し、よろめきへ暗黙追加しない。

## 反撃しない攻撃

- Bossを含む範囲攻撃
- 直線突進の通過ダメージ
- 地形、罠、毒、炎上、衝突によるダメージ
- 警戒射撃など別の反応攻撃
- 反撃そのもの
- 対象を直接指定しないObject・装置効果

将来、範囲攻撃へ反撃できる専用スキルを作る場合は効果文へ明記する。通常の反撃準備を流用しない。

## Objectiveと勝敗

反撃は元攻撃と同じRoot Action Batch内で処理する。

```text
元攻撃解決
-> 反撃条件判定
-> 反撃解決
-> 双方の戦闘不能Event確定
-> Objective更新
-> 敗北判定
-> 勝利判定
```

反撃で敵を倒した場合、通常の`UnitDefeated`として敵全滅・指揮官撃破・条件付き撃破へ加算する。
「能動攻撃で撃破」のように明示された副目標には加算しない。反撃した槍兵が状態ダメージなどで
同じBatch中に倒れた場合も、全Event確定後に敗北優先で判定する。

## UI

- 攻撃予測に通常反撃欄を常設しない
- 発動可能な反応がある時だけ「反応: 反撃準備」を表示
- 射程外など発動しない場合は非表示にせず、Unit詳細で不成立理由を確認可能にする
- 反撃発生時は1件の戦闘メッセージを出し、通常行動選択画面を開かない

## 受入条件

- スキルなしの隣接Unit同士では反撃しない
- 反撃準備は行動済みでも1戦闘1回だけ発動する
- 射程外へ押し出された場合は発動せず使用回数を維持する
- よろめき中でも現在地から反撃できる
- Boss範囲攻撃、毒、警戒射撃へ反撃しない
- 反撃撃破が敵全滅Objectiveへ一度だけ加算される
- 反撃から反撃が連鎖しない
