## 地域概要

沈黙した監視所群から得た旧補給図は、前線施設へ石材と鉄鉱石を送っていた採石場を示していた。
採石場は崩落後に放棄され、採掘設備、搬出路、坑道支柱が損傷している。回収団の残党と採掘物を
縄張りにする穿岩獣が入り込み、無計画な採掘によってさらに崩落が進んでいる。

深部の記録には、通常鉱脈とは異なる計測反応が残されている。これは無限の魔力鉱石や未知文明の
動力源ではなく、局所的にAetherへ反応する鉱石層である。訓練を受けた術者と器具なしでは利用
できず、採取量を増やしても一般兵が魔法を使えるようにはならない。

## 開拓できない理由

- 搬入口が崩落し、荷車と建築材を運び出せない
- 巻上機と搬出路が壊れ、採掘物を地上へ上げられない
- 坑道支柱が弱り、無計画な破壊で戦場全体が崩落する
- 穿岩獣が振動源と採掘作業員を襲う
- 異常鉱脈を通常の鉄鉱脈と同じ方法で掘ると反応と崩落が拡大する

地域攻略は敵全滅だけではなく、**搬入口、搬出設備、坑道、安全な採掘手順、深部封鎖の五つを
確立すること**を意味する。

## 地域の役割

- 破壊可能Object、防護対象、装置操作、崩落予告を本格導入する
- 重装兵、辺境工兵、槍兵、戦闘魔導士の異なる役割を試す
- 全12兵種を使用可能にし、以後は新兵種ではなく横方向の戦術解放へ移る
- 石材、鉄鉱石、木材の恒久供給を確保し、辺境集落へ発展可能にする
- 魔法を万能解決にせず、危険な現象を計測・制御する専門技能として示す

## 地点構成

```text
1. 崩落した搬入口
        ↓
2. 砕石段丘
        ↓
    キャンプI
        ↓
3A. 旧採掘坑 ──┐
                 ├─ 4. 灰鉄鉱脈
3B. 巻上機区画 ─┘
        ↓
    キャンプII
        ↓
5. 崩落核
        ↓
地域攻略・黒水低湿地解放
```

- 地点1と2は順番固定
- キャンプI後は旧採掘坑と巻上機区画のどちらを先に処理してもよい
- 地点3Aまたは3Bを一つ確保すれば灰鉄鉱脈へ進める
- 両方を確保すると地点4と5が軽量化される
- キャンプIIは地点4後に利用可能
- 遠征は常に地点1から始まり、キャンプから恒久再開しない

## 周回と短縮

標準4周、慎重な攻略または副目標回収を含めて5周を想定する。

| 恒久成果 | 次回以降の効果 |
|---|---|
| 搬入口確保 | 地点1を戦闘なしで通過 |
| 砕石路整備 | 地点2を戦闘回避。未取得通常素材だけ再採取可能 |
| 旧採掘坑補強 | 地点4・5の崩落予告を1ラウンド早く表示 |
| 巻上機復旧 | キャンプIで1遠征1回、持込品1個を倉庫から補給可能 |
| 安全採掘手順 | 地点4を再訪時に非戦闘採取へ変更 |
| 深部封鎖 | 地域攻略。再訪は素材回収または訓練任務のみ |

想定進行:

1. 第1周は地点2後に帰還し、搬入口と砕石路を確保
2. 第2周は旧採掘坑または巻上機区画を確保して帰還
3. 第3周は残る区画と灰鉄鉱脈を調査し、異常鉱脈記録を持ち帰る
4. 第4周は復旧設備と戦闘魔導士の情報を利用して崩落核を封鎖
5. 必要なら第5周で副目標、高品質素材、未回収記録を取得

早期に続行できる場合は1〜3周での攻略も許可し、人工的な周回ロックを設けない。

## 地形生成

全戦闘は3行×8列。24マスすべてを遠征Seedから生成し、初期配置ユニットのマスだけ必要時に
通行可能床へ補正する。

### 共通生成率

| 地形 | 生成目安 | 効果 |
|---|---:|---|
| 採石床 | 35〜50% | 移動コスト1 |
| 砕石 | 15〜25% | 移動コスト2。辺境斥候は1 |
| 軌道床 | 10〜15% | 移動コスト1。ノックバック距離+1 |
| 支柱床 | 5〜10% | DEF+2。隣接支柱が壊れると通常床化 |
| 崩落予告床 | 5〜15% | 次のRound終了時に瓦礫化。予告中は通行可能 |
| 岩壁・大型瓦礫 | 5〜10% | 通行不可。破壊可能Objectの場合は耐久を持つ |

- 茂み、浅瀬、薬草地点は生成しない
- 通行不能地形で3行すべてを封鎖しない
- 両陣営間に最低1本の通行可能経路を保証する
- 初期配置、脱出、装置、増援、必須Objectiveマスを通行不能にしない
- 崩落予告は戦闘前または1ラウンド前に表示し、即時の非公開崩落を起こさない

### 採石場Object

| Object | 耐久 | ルール |
|---|---:|---|
| 小型瓦礫 | 6 | 破壊すると通行可能。重装兵と工兵は与ダメージ+3 |
| 坑道支柱 | 8 | 破壊されると指定2〜3マスへ崩落予告。味方の通常攻撃対象にしない |
| 巻上機 | 12 | 操作・防衛対象。耐久0でその戦闘の補給副目標失敗 |
| 鉱石箱 | 5 | 敵が隣接行動終了を2回行うと持ち去られる |
| 封鎖杭 | 10 | 最終戦で設置・防衛する。耐久0で再設置が必要 |

## 敵勢力と戦術

### 回収団採掘班

- 鉱石箱、巻上機、脱出路を優先する
- 鉱石を確保すると左端ではなく敵側退路へ撤退する
- 支柱を無差別に破壊せず、退路確保または遠征隊分断に利用する
- 主な兵種: 斧兵、弓兵、工兵崩れ

### 穿岩獣

採石場の振動と掘削音に反応する大型節足動物。魔物ではなく、この地域に適応した危険な野生
生物として扱う。

- 最も近い振動源、孤立ユニット、低HP対象の順に狙う
- Object操作や採掘を行ったユニットの脅威評価を上げる
- 人間の装置や任務目的は理解しない
- 防御は高いがRESは低い

### 灰殻穿岩虫

最終地点の大型個体。異常鉱脈から生まれた存在ではなく、深部の空洞を縄張りにしていた個体。
鉱脈反応はその移動で露出した地層から観測される。

### 探索3択

| `[辺境工兵]` 軌道を切り替える | 辺境工兵 | 落石方向を1回だけ指定可能、敵4体 |

### 任務

- 主目的: 右側の搬出標識で味方が行動終了
- OR条件: 敵全滅後に搬出標識を確保
- 副目標: 鉱石箱を持ち去らせない
- 敗北条件: 部隊全滅、鉱石箱2個を両方失う

### キャンプI

## 3A. 旧採掘坑

坑道には生存した採掘作業員と、崩落寸前の支柱が残る。敵を倒す速度より、退路と支柱を守る
判断が重要になる。

### 探索3択

| 選択 | 条件 | 効果 |
|---|---|---|
| 作業員の声を追う | なし | 作業員2人を護衛、標準敵4体 |
| 支柱を先に確認する | なし | 作業員1人、崩落予告を常時表示 |
| `[古参守備兵]` 退路を封鎖する | 古参守備兵 | 増援なし、味方初期配置は左2列 |

### 任務

- 主目的: 作業員1人以上を脱出させる
- 副目標: 作業員2人とも脱出
- 副目標: 坑道支柱2本のうち1本以上を保全
- 敗北条件: 部隊全滅、全作業員の撤退
- 敵: 穿岩獣3、回収団弓兵1。2ラウンド目に穿岩獣1体の予告増援

### 報酬

- 勝利: 木材2、石材1
- 全員救出: 採掘技術記録
- 支柱保全: 木材+1
- 恒久成果: `old_mine_reinforced`

採掘技術記録は失敗しても地点5の作業台帳から代替取得でき、永久取得不能にしない。

## 4. 灰鉄鉱脈

通常の鉄鉱脈に混じって計測器へ不規則な反応を返す地層が露出している。イリエン・
ヴェイルリーチは、敵を抑えながら反応地点を3か所測定するよう求める。

### 探索3択

| 選択 | 条件 | 効果 |
|---|---|---|
| 通常鉱脈だけを採る | なし | 測定1か所、敵4体、鉄鉱石+1 |
| 異常反応を調べる | なし | 測定3か所、敵5体、崩落予告あり |
| `[戦闘魔導士候補]` 反応を固定する | 地点3Aまたは3B確保 | 測定2か所、崩落予告を1回無効化 |

この地点ではイリエンを操作可能なゲストユニットとして扱う。HP0では死亡せず撤退し、その遠征
では異常鉱脈記録を取得できない。次回再挑戦できる。

### 任務

- 主目的: 指定測定地点をすべて操作し、左側退路へ戻る
- 副目標: イリエンを撤退させない
- 副目標: 鉱石箱1個を確保
- 敗北条件: 部隊全滅、全測定器破壊
- 敵: 穿岩獣4。異常反応ルートでは大型穿岩獣1を追加

### 報酬と加入候補

- 勝利: 鉄鉱石2、石材1
- 通常鉱脈ルート: 鉄鉱石+1
- 測定完了かつイリエン生存: 異常鉱脈記録、`mage_recruit`
- 鉱石箱確保: 高品質鉄材1
- 恒久成果: `ashiron_vein_surveyed`

異常鉱脈記録は魔法を一般化せず、戦闘魔導士の加入と専門設備の安全運用条件にだけ使う。

### キャンプII

地点4後に利用可能。異常鉱脈記録と加入候補を安全帰還で確定すると、集会所にイリエンの加入
会話が追加される。遠征別棟は地域攻略後に解放するため、この時点ではゲスト参加のみ。

## 5. 崩落核

深部空洞で灰殻穿岩虫が支柱と岩壁を壊し続けている。遠征隊は個体を深部へ退かせ、二つの封鎖杭を
設置して採石区域から切り離す。

### 探索3択

| 選択 | 条件 | 効果 |
|---|---|---|
| 支柱を守りながら進む | なし | 支柱2本、標準ボス戦 |
| 古い火割り溝を使う | 燃焼油1 | 岩壁1個を熱して破砕、ボスHP-6、崩落予告+1列 |
| `[戦闘魔導士]` 振動を逸らす | イリエン加入候補確定 | 最初の潜行攻撃を無効、測定地点1個を防護対象に追加 |

### 主目的

次のAND条件をすべて達成する。

1. 灰殻穿岩虫のHPを0にして深部へ撤退させる
2. 上下2か所の封鎖杭を設置する
3. いずれかの封鎖杭を耐久1以上で戦闘終了まで守る

副目標:

- 坑道支柱を1本以上保全
- 味方戦闘不能者0
- 8ラウンド以内に封鎖完了

敗北条件:

- 部隊全滅
- 封鎖杭を設置できる2地点が両方永久崩落
- 10ラウンド終了時に主目的未達成

## 地域ボス 灰殻穿岩虫

### 基本値

| 項目 | 値 |
|---|---:|
| 占有 | 1マス |
| HP | 56 |
| STR | 9 |
| MAG | 0 |
| DEF | 8 |
| RES | 2 |
| MOV | 3 |

### 行動

#### 潜行突進

- 1ラウンド前に出現予定マスと通過列を表示
- 次のEnemy Phase開始時に直線3マスを移動
- 通過したユニットへSTR+3の物理ダメージ
- 支柱へ衝突すると支柱へ4ダメージを与えて停止
- 対象の移動後に追尾しない

#### 岩殻防御

- 潜行から出た直後以外はDEF+2
- 潜行突進後、次のEnemy Phase開始までDEF補正を失う
- 防御低下は通常ボス補正のDEF-2を適用

#### 崩落誘発

- HP50%以下で一度だけ使用
- 空いている2マスを崩落予告床にする
- 初期配置、封鎖杭設置地点、唯一の通路は対象外
- 次のRound終了時に瓦礫化

### 行動優先順位

1. 予告済み潜行突進を実行
2. HP50%以下で崩落誘発が未使用なら使用
3. 封鎖杭を操作中の味方へ潜行突進を予告
4. 隣接攻撃で戦闘不能にできる味方を攻撃
5. 最も近い操作・採掘実行者へ移動
6. それ以外は封鎖地点間を妨害する位置へ移動

完全な行動不能は無効。毒、炎上、移動低下、防御低下、よろめきは`status_effects.md`の通常ボス
補正を適用する。

### ボス戦報酬

| 条件 | Pending報酬・進行 |
|---|---|
| 主目的達成 | 鉄鉱石2、石材2、木材1、穿岩殻1 |
| 支柱1本以上保全 | 木材+1、重装加工記録を未取得なら追加 |
| 戦闘不能者0 | 高品質鉄材1 |
| 8ラウンド以内 | 燃焼油1 |
| 初回地域攻略 | 採掘技術記録を未取得なら作業台帳から追加 |

初回地域攻略時、重装加工記録または異常鉱脈記録が未取得なら、封鎖後の設備調査で未取得分だけを
追加する。重装加工記録は破損支柱の補強記録、異常鉱脈記録はイリエンが回収した測定器ログを
取得元とする。同じ記録は報酬台帳により重複追加しない。各地点で先に取得すれば、地域攻略前の
安全帰還から施設情報と加入会話を早期に利用できるため、副目標の価値は残る。

## 地域攻略と拠点接続

安全帰還時に次を処理する。

- `ashiron_quarry_secured`をDiscovery Registryへ登録
- 採掘技術記録で辺境集落への発展条件を満たす
- 重装加工記録で鍛冶場第2段階の前提を解放
- 異常鉱脈記録で遠征別棟と戦闘魔導士加入条件を満たす
- イリエン・ヴェイルリーチの加入会話後、全12兵種を使用可能にする
- 鍛冶場と工房の第2段階を研究可能にする
- 黒水低湿地を次の遠征先へ追加

辺境集落への発展は自動ではない。安全帰還後、拠点画面で採掘技術記録を確認し、石材4、
鉄鉱石4、木材4を消費して確定する。重要発見は消費しない。

## 最低保証報酬

探索ルートや任意副目標にかかわらず、5地点の主目的を達成して安全帰還した場合、地域全体で
最低限次を取得できる。

| 資産 | 最低保証 |
|---|---:|
| 鉄鉱石 | 6 |
| 石材 | 6 |
| 木材 | 4 |
| 燃焼油 | 0（探索・副目標で追加） |
| 採掘技術記録 | 1 |
| 重装加工記録 | 1 |
| 異常鉱脈記録 | 1 |
| 穿岩殻 | 1 |

- 石材4、鉄鉱石4、木材4を残せば必ず辺境集落へ発展できる
- 素材を製作へ使った場合は、確保済み地点の非戦闘再採取で補充できる
- 重要発見は売却・消費・重複取得できない
- 副目標報酬は高品質鉄材、追加素材、早期解放に限定する

## 実装順

1. 5段階と地点3の分岐データ
2. 破壊可能Objectと崩落予告
3. 搬入口と砕石段丘の主・副目標
4. 護衛、脱出、巻上機操作
5. 灰鉄鉱脈の測定とゲストユニット
6. 灰殻穿岩虫の予告行動と封鎖杭
7. 周回短縮、最低保証、重複防止
8. イリエン加入、全12兵種、辺境集落、黒水低湿地解放

## 実装時の不変条件

- 魔法や異常鉱脈を無限資源、汚染、人格変化、蘇生へ接続しない
- 採掘量だけで兵種能力値を一律上昇させない
- Object破壊と崩落を予告なしの即時敗北にしない
- 必須進行を工兵、重装兵、戦闘魔導士の編成へ依存させない
- 兵種専用選択は有利な代替手段とし、通常ルートを残す
- イリエンの一度の撤退で永久加入不能にしない
- 全12兵種解放後、新兵種追加を主要成長にしない
- 地域進行成果と通常素材を同じ消費可能データに入れない
