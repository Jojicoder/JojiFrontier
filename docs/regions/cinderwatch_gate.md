## 地域概要

灰枝の森を越えて街道を確保した遠征隊は、シンダーウォッチの外側に連なる6つの旧施設を調査
する。灰嵐以降、監視所からの信号は途絶え、物資庫と兵舎は元守備兵、脱走兵、回収団に利用
されている。

調査を進めると、信号網は事故ではなく、元守備隊長の命令で停止されたことが分かる。彼は救援が
来ないと判断し、生存者と物資を集め、外側へ人間の位置を知らせないために信号を切った。

遠征隊は街道と監視網の再建を優先する立場から元守備隊と対立する。最終的に信号塔を復旧すると、
地図にない遠方から一度だけ応答が返り、灰鉄採石場方面の古い中継経路が判明する。

## 開拓できない理由

この地域を確保しない限り、灰枝の森から先へ人員と物資を継続輸送できない。

- 外門が瓦礫と回収団によって封鎖され、荷車が通れない
- 灰道の監視情報が失われ、敵配置と安全路を把握できない
- 物資庫と旧兵舎が使えず、長距離遠征の補給と伝令が成立しない
- 信号塔が停止し、March Line側と遠征隊が互いの危険を通知できない
- 元守備隊が信号再開を阻止しており、設備だけ直しても運用を再開できない

したがって地域攻略は領土征服ではなく、**外門、監視、補給、伝令、信号の五機能を復旧すること**
を意味する。元守備隊長の撃破だけでは地域攻略にならない。

## 地域の役割

- 野生動物中心だった灰枝の森から、人間勢力との戦闘へ移行する
- 敵全滅以外に地点確保、防衛、護衛、装置操作を導入する
- 工兵、伝令騎兵、旗手を早期加入させる
- 偵察、野戦医療、信号、軍旗の重要発見を拠点へ接続する
- 6地点と2回のキャンプで、続行と安全帰還の判断を本格化する
- 元守備隊を単純な悪人にせず、開拓と防衛の立場の違いを示す

## 地点構成

```text
1. シンダーウォッチ外門
        ↓
2. 灰道の監視所
        ↓
   キャンプI
        ↓
3A. アイアンウォッチ物資庫
       ↘
         合流
       ↗
3B. 旧兵舎
        ↓
   キャンプII
        ↓
5. 信号塔下層
        ↓
6. 最後の信号
        ↓
地域攻略・灰鉄採石場解放
```

- 地点2と地点4の後にキャンプを置く
- キャンプI後は物資庫と旧兵舎のどちらを先に攻略してもよい
- 先に攻略した地点から安全帰還できるが、両地点を確保するまでキャンプIIは恒久利用できない
- 物資庫を先に確保すると補給を得られ、旧兵舎を先に確保すると物資庫の増援を予告できる
- 次回遠征も地域入口から開始する
- 確保済みの前半地点は安全通過でき、後半へHPとアイテムを温存できる
- 確保済み地点の初回報酬と重要発見は再取得できない
- 以前の地点へ再訪する場合は訓練用の縮小報酬だけを使用し、本編進行を重複させない

## 周回による経路開拓

この地域は5周攻略を標準とし、6周目は取り逃した副目標や追加素材の回収に使う。各遠征は
シンダーウォッチ外門から始まるが、安全帰還で確定した開拓成果により前半の必須戦闘を
安全通過へ変えられる。

| 地点 | 経路確保条件 | 次回以降の軽量化 |
|---|---|---|
| シンダーウォッチ外門 | 記録箱を回収して勝利 | 開門手順が判明し、戦闘なしで外門を通過 |
| 灰道の監視所 | 監視地点を2か所確保 | 敵と危険地形を事前表示。安全路なら戦闘回避 |
| アイアンウォッチ物資庫 | 工兵救出と工具区画確保 | 物資庫を補給地点化。1遠征1回、キャンプで野戦治療キット1個を補充 |
| 旧兵舎 | 伝令路を確保して防衛成功 | 伝令騎兵が先導し、地点4の通常戦闘を省略 |
| 信号塔下層 | 制御盤を両方保全 | ボス戦の増援を事前表示し、副信号機ルートを使用可能 |
| 最後の信号 | 地域攻略 | 再訪時は訓練戦または未達成副目標だけを選択 |

想定進行:

1. 第1周は地点2後のキャンプから帰還し、外門と監視所を踏査する
2. 第2周は外門を安全化し、物資庫で工兵を救出して補給地点化する
3. 第3周は残る物資庫または旧兵舎を攻略し、補給と伝令路の両方を確保する
4. 第4周は工兵・伝令騎兵を編成し、信号塔下層の制御盤を保全して帰還する
5. 第5周は開通路、補給、副信号機を利用して元守備隊長を攻略する
6. 必要なら第6周で軍旗記録、高品質鉄材、未達成副目標を回収する

予定より早い周回でさらに先へ進むことは制限しない。到達できるプレイヤーへ人工的な周回制限を
設けず、キャンプで帰還する価値と、続行して早期攻略を狙う価値を併存させる。

## 地形生成

論理盤面は全戦闘で3行×8列。24マスすべてを遠征Seedから生成し、固定配置図を使用しない。

### 共通地形

| 地形 | 生成目安 | 効果 |
|---|---:|---|
| 床 | 40〜55% | 移動コスト1 |
| 灰地 | 10〜20% | 移動コスト2。辺境斥候のみ1 |
| 瓦礫 | 15〜25% | 移動コスト2 |
| 監視床 | 5〜15% | 防御+2 |
| 障害物 | 5〜10% | 通行不可。1列につき最大1マス |

- 沈黙した監視所群では茂みを生成しない
- 浅瀬と薬草地点もこの地域の通常生成には含めない
- 初期配置されたユニットのマスだけ、通行不能地形を床へ変更する
- 初期配置範囲の未占有マスは生成結果を維持する
- 障害物で3行すべてを封鎖しない
- 両陣営間に最低1本の通行可能経路を保証する
- 任務目標、増援、装置、脱出予定マスには障害物を置かない

### 地点別補正

| 地点 | 地形傾向 | 必須要素 |
|---|---|---|
| 外門 | 瓦礫と障害物が多い | 中央4列に突破可能な経路1本 |
| 監視所 | 監視床が2〜3マス | 右側に踏査地点1マス |
| 物資庫 | 床と障害物が多い | 物資箱2個、医療区画1マス |
| 旧兵舎 | 灰地と瓦礫が均等 | 護衛対象、脱出マス1〜2個 |
| 信号塔下層 | 監視床と装置が多い | 信号制御盤2個 |
| 最後の信号 | 監視床3マス以上 | 主信号機、ボス初期位置、増援口 |

## 敵勢力

### 回収団

放棄施設から物資を持ち出す武装集団。遠征隊と同じ資源を狙うが、組織的な軍隊ではない。

- 低HPの物資箱や脱出路を優先
- 不利になると戦利品を持って撤退しようとする
- 隊形維持や装置操作は行わない
- 主な兵種: 盗賊、弓兵

### 脱走兵

元守備隊から離脱した兵士。生存と食料を優先し、必ずしも元守備隊長に忠誠を持たない。

- 孤立した味方を狙う
- HPが30%以下になると撤退路を評価する
- 降伏または撤退した敵は撃破報酬を落とさないが、副目標の生存者確保へ影響できる
- 主な兵種: 槍兵、斧兵

### 元守備隊

旧信号塔を維持している訓練済みの兵士。信号再開を阻止するため、地点防衛と通路封鎖を行う。

- 監視床、装置、狭路を優先して守る
- 全員が同じ標的へ突撃せず、前衛と射撃役を分ける
- 元守備隊長の命令範囲内では撤退しない
- 主な兵種: 古参守備兵、監視弓兵、槍兵、行軍隊長

人間系の戦闘不能は死亡を意味しない。撤退、降伏、負傷による戦線離脱を基本表現とする。

### あらすじ

森から続く街道の先で、外門は崩れた石材と荷車で塞がれている。回収団が門内の旧記録を燃やし、
鉄具だけを運び出そうとしている。

### 探索選択

| 選択 | 条件 | 戦闘への影響 | 報酬への影響 |
|---|---|---|---|
| 正面の瓦礫を越える | なし | 標準配置、敵4体 | 標準報酬 |
| 崩れた排水路を進む | なし | 全員HP-2、敵3体、左3列配置 | 石材-1 |
| `[重装兵]` 門材を押し退ける | 重装兵 | HP消耗なし、障害物1個除去、敵4体 | 石材+1 |

### 任務

- 副目標: 右側の記録箱を敵より先に確保する

### 敵編成

- 達成方法: 敵全滅、または中央の監視床を2ラウンド維持

| 地図作成 | 石材1 |
| 斥候ルート | 次の地点の完全情報 |

### キャンプI

地点2攻略後に設営する。ここから安全帰還すると地点1〜2の踏査成果と、達成済みの

### あらすじ

物資庫では、負傷した工作兵が回収団に道具の修理を強制されている。医療品と工具は別区画にあり、

## 5. 信号塔下層

### あらすじ

信号塔の下層では、副信号機と主信号機の接続が切られている。元守備隊は装置を壊さず、再起動
できない状態へ分解していた。遠征隊は敵を抑えながら、二つの制御盤を復旧する。

### 探索選択

| 選択 | 条件 | 戦闘への影響 | 代償 |
|---|---|---|---|
| 一つずつ制御盤を直す | なし | 標準戦闘 | なし |
| 敵を先に排除する | なし | 敵全滅後に操作 | 6ラウンド制限 |
| `[辺境工兵]` 副信号機を迂回接続 | 辺境工兵 | 必要操作盤が2個から1個 | 工具部品1を消費 |

### 任務

- 主目的: 2つの制御盤を操作して接続を復旧
- OR条件: 敵全滅後、残った制御盤を操作
- 副目標: 軍旗保管箱を確保
- 敗北条件: 部隊全滅、主制御盤破壊、明示されたルートでの6ラウンド超過

### 敵編成

- 古参守備兵1
- 監視弓兵2
- 槍兵2
- 制御盤1個目の操作後、反対側の増援口から斧兵1体

### 報酬

| 条件 | 未確定報酬・進行 |
|---|---|
| 勝利 | 鉄材1、旧軍備1 |
| 制御盤を両方保全 | 信号技術資料 |
| 軍旗保管箱確保 | 軍旗記録 |
| 工兵ルート | 増援なし |

信号技術資料は工房の帰還信号技術、軍旗記録は旗手の加入条件と訓練所の専門訓練へ接続する。

## 6. 最後の信号

### あらすじ

主信号機の前で元守備隊長が遠征隊を待つ。彼は救援が来ないまま部下を失い、残った人員を守る
ために外縁の監視網を切った。信号再開は外側へ人間の位置を知らせる危険な行為だと主張する。

遠征隊長は、信号と街道を失ったままでは孤立が繰り返されるとして復旧を求める。交渉は決裂し、
主信号機を巡る戦闘になる。

### 探索選択

| 選択 | 条件 | 戦闘への影響 | 報酬への影響 |
|---|---|---|---|
| 正面から復旧を宣言 | なし | 標準ボス戦 | 標準 |
| 副信号機を先に点灯 | 信号塔下層攻略 | 3ラウンド目の敵増援なし | 主信号機の耐久-3 |
| `[行軍隊長]` 元守備兵へ退路を示す | 行軍隊長 | 護衛2体がHP50%以下で撤退可能 | 旧軍備+1 |

### 主目的と敗北条件

- 主目的: 元守備隊長を戦闘不能にして撤退させる
- 副目標: 主信号機を破壊されずに勝利
- 副目標: 元守備兵を2人以上撤退・降伏させる
- 副目標: 味方戦闘不能者0
- 敗北条件: 部隊全滅、主信号機破壊

### 主信号機の耐久

- 基本耐久は12
- 「副信号機を先に点灯」を選んだ場合は耐久9で開始
- 指定された敵増援は隣接時に「妨害工作」を使用し、固定4ダメージを与える
- 通常の単体攻撃は主信号機を対象にできない
- 主信号機へ影響すると明記された範囲攻撃と衝突は耐久を減らす
- 辺境工兵の設備修復を使用できる場合は3回復し、最大耐久を超えない
- 耐久0で即座に敗北する

## 地域ボス 元守備隊長

正式名は人物設定のWorld Bible提案まで未確定。表示名は「元守備隊長」とする。

### 基本値

| 項目 | 値 |
|---|---:|
| 兵種 | 行軍隊長 |
| HP | 32 |
| STR | 9 |
| MAG | 1 |
| DEF | 7 |
| RES | 4 |
| MOV | 4 |
| 武器 | 鉄の剣、威力5、射程1 |

現行試作の行軍隊長基礎値にHP+10、DEF+2、STR+2を加えた地域ボス用能力とする。

### 固有行動

#### 射線命令

- 戦闘中1回
- 使用1ラウンド前に対象行を赤い縞で予告
- 次の敵Phase開始時、対象行にいる元守備隊の射撃役が射程内の味方を優先攻撃
- 射撃役が戦闘不能、射程外、行動不能なら攻撃は発生しない
- ボス本人の追加攻撃は発生しない

#### 防衛隊形

- HP50%以下で一度だけ発動
- ボスと隣接する元守備兵のDEF+2
- 次の敵Phase終了まで
- 一時強化上限+4に従う

#### 信号封鎖

- 主信号機に隣接している間、味方は主信号機を操作できない
- ボスを押し出す、誘導する、戦闘不能にすることで解除
- 主信号機そのものから攻撃は発生しない

### 行動優先順位

1. 予告済み射線命令を解決
2. HP50%以下で防衛隊形が未発動なら使用
3. 主信号機へ敵が隣接していれば、その敵を排除
4. 隣接する味方が2人以上攻撃範囲にいれば通常攻撃
5. 射線命令が未使用なら、最も多くの味方がいる行を予告
6. それ以外は主信号機から2マス以内を維持して移動・攻撃

ボスは挑発の対象評価を上げるが、予告済み行動と主信号機防衛を放棄しない。

- 元守備隊長1
- 古参守備兵1
- 監視弓兵2
- 槍兵1
- 標準ルートでは3ラウンド目に斧兵1体増援

## 地域攻略結果

元守備隊長は死亡せず、負傷して撤退または拘束される。信号塔を復旧すると、地図に記録されて
いない遠方から一度だけ応答が返る。応答内容は位置確認に必要な短い信号だけで、外側の正体を
説明しない。

応答方向と旧補給図を照合すると、灰鉄採石場を経由する中継路が判明する。

```text
沈黙した監視所群を攻略
-> 信号網を復旧
-> 遠方から一度だけ応答
-> 灰鉄採石場への経路判明
-> 第3地域を解放
```

安全帰還時:

- 偵察資料を登録し、司令所「偵察網」を解放可能
- 野戦医療記録を登録し、診療所「野戦医療」を解放可能
- 信号技術資料を登録し、工房「帰還信号技術」を解放可能
- 軍旗記録を登録し、旗手と専門訓練を解放可能
- 辺境工兵、伝令騎兵、旗手の加入条件を満たした人物を宿舎へ追加
- 灰鉄採石場を遠征先へ追加

## 地域の最低保証報酬

探索ルートや副目標にかかわらず、6地点を攻略して安全帰還した場合は最低限次を保証する。

| 資産 | 最低保証 |
|---|---:|
| 鉄材 | 5 |
| 石材 | 3 |
| 旧軍備 | 3 |
| 信号機の中核部品 | 1 |
| 偵察資料 | 1 |
| 野戦医療記録 | 1 |
| 信号技術資料 | 1 |
| 軍旗記録 | 1 |

- 重要発見を副目標失敗だけで永久取得不能にしない
- 初回失敗時は後続地点または再挑戦で代替取得できる
- 全重要発見を一度の遠征で取得する必要はない
- 最低保証だけで次地域解放と基礎施設解放が可能
- 高品質鉄材、追加旧軍備、追加石材は副目標と危険ルートの報酬にする

重要発見を対応地点で取得できなかった場合、最終地点の勝利後調査から未取得分だけを
回収できる。偵察資料は守備隊長の指揮地図、野戦医療記録は保管台帳、信号技術資料は
主信号機の記録部、軍旗記録は指揮箱を取得元とする。これは取り逃し防止用の代替取得で、
同じ重要発見を重複して追加しない。途中で取得した場合はキャンプ帰還後から早期解放へ
利用できるため、各地点の副目標を達成する価値は残る。
