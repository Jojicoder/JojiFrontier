## 地域概要

黒水低湿地を越えた先に、旧街道が高原へ登る。風雨で路肩と中継設備が崩れ、荷車と負傷者を同じ
速度で運べない。残った伝令所は、街道を独占する武装輸送団「高原運び手」に押さえられている。

高原運び手は無差別な略奪者ではない。公的補給が途絶えた後、自分たちで道を維持し、危険な輸送を
請け負ってきた。しかし通行料、救援順、荷物の優先権を一方的に決め、弱い隊を置き去りにする。
遠征隊は彼らを全滅させるのではなく、誰でも使える伝令路と救援規則を確立する。

## 開拓できない理由

- 強風と崩落で安全な登り道が毎回変わる
- 中継路と吊り橋が壊れ、荷車と負傷者を同時に運べない
- 風見台が使えず、突風と敵の接近を事前に知らせられない
- 武装輸送団が伝令所を占有し、通行と救援の優先順位を独占している
- 高原を越えた先の旧辺境集落へ継続的な伝令を送れない

地域攻略は、**登り道、中継路、風見台、輸送規則、伝令所**を公共の街道機能として復旧することを
意味する。

## 地域の役割

- 高MOV、再移動、悪路移動、射程、護衛を別々の強みとして検証する
- 分散初期配置と合流判断を導入する
- 風による強制移動を予告付き地形として扱う
- 荷物と負傷者のどちらを優先するか選ばせる
- 機動訓練、騎兵装備、輸送情報を拠点へ接続する
- 全Unitへの恒久MOV上昇は行わない

    キャンプI
        ↓

                    ├─ 攻略順を選択
4. 分断された輸送隊 ─┘

        ↓

        ↓

- 地点1・2は順番固定
- 地点3・4はどちらを先に攻略してもよい
- 片方の成果だけ持ち帰ることもできる
- 地点5へ進むには地点3・4の両方を確保する
- キャンプは地点2と4の進行段階後
- 遠征は常に地点1から開始する

## 周回と短縮

標準5周、早期続行4周、副目標回収6周を想定する。

| 恒久成果 | 次回以降の効果 |
|---|---|
| 登り口標識 | 地点1を戦闘なしで通過 |
| 中継橋補修 | 地点2を安全通過。キャンプIでHPが最も低い生存者を3回復 |
| 風見台復旧 | 全地点の強風方向と発生Roundを戦闘前に表示 |
| 輸送隊合流 | 護衛対象の初期MOV+1、増援位置を公開 |
| 荷車道確保 | 地点5を安全通過し、再採取素材を運搬可能 |
| 伝令所開放 | 地域攻略。再訪は輸送任務または訓練戦のみ |

|---|---:|---|

| 砕石斜面 | 15〜25% | 移動コスト2。辺境斥候は1 |
| 稜線 | 10〜15% | 移動コスト1、DEF+1 |

### 強風ルール

- 戦闘開始時に風向きと最初の発生Roundを表示する
- 発生1ラウンド前から対象マスへ矢印を表示する
- Round終了時、強風帯上のユニットを矢印方向へ1マス移動する
- 移動先が断崖、盤外、Unit、通行不能Objectなら移動せず、固定2ダメージを受ける
- 重装兵の重量装甲は強制移動距離1を0にするため移動しない。衝突ダメージも受けない
- 防護板は風を止めないが、移動先を塞ぐObjectとして扱う
- 強風でObjective達成、装置操作、脱出を自動成立させない
- 初期配置、脱出、必須操作マスを強風帯にしない
- 戦闘ごとに風向きは固定し、Round途中でランダム変更しない

### 分散配置

- 指定戦闘では4人を上段2人、下段2人に分けて開始する
- 各組に最低1本の合流経路を保証する
- 同じUnitを複数地点へ置かない
- 初期配置マスだけ通行可能な高原道へ補正する
- 分散配置を選ばない探索ルートも必ず用意する

## 敵勢力

### 高原運び手

- 伝令、荷車、脱出路を戦術目標として理解する
- 騎兵は攻撃後に安全圏へ再移動する
- 弓兵は稜線を優先し、護衛対象への射線を作る
- 槍兵は狭い橋と登り道を封鎖する
- HP30%以下で撤退を評価し、死亡ではなく戦線離脱として扱う

### 断崖の野盗

高原運び手とは別の小規模略奪集団。荷物を奪って盤外へ逃げることを優先し、伝令所の防衛や
街道維持には関心を持たない。

- 荷物箱を確保したUnitは最短の敵側退路へ移動
- 孤立した護衛対象を優先
- 主な兵種: 斧兵、弓兵、軽装剣士

## 6. 高原伝令所

武装輸送団の隊長は、遠征隊が道を維持できる保証がないとして伝令所の引き渡しを拒む。戦闘は
隊長を殺すためではなく、公開伝令路を実際に機能させられることを示すために行う。

### 探索3択

| 選択 | 条件 | 効果 |
|---|---|---|
| 隊長へ引き渡しを要求 | なし | 標準ボス戦 |
| 先に伝令を走らせる | 高原街道図 | 味方伝令1人を追加、敵増援+1 |
| `[行軍隊長]` 共同運用を提案 | 行軍隊長 | 敵2人がHP50%以下で撤退可能、伝令所耐久+4 |

### 主目的

次のいずれかを達成する。

- 高原運び手の隊長を戦闘不能にして撤退させる
- 味方伝令を右端の伝令箱へ到達させ、伝令所を2ラウンド維持する

副目標:

- 伝令所耐久を8以上残す
- 高原運び手を2人以上撤退・降伏させる
- 味方戦闘不能者0

敗北条件:

- 部隊全滅
- 伝令所耐久0
- 味方伝令が撤退し、隊長撃破も不可能になった場合

### 固有行動

#### 通り抜け攻撃

- 直線で2マス以上移動した後だけ使用可能
- 通常攻撃後、敵から離れる方向へ最大2マス再移動
- Zone of Control、Unit、通行不能地形を無視しない
- 再移動先がなければ通常攻撃だけ行う

#### 迂回命令

- 戦闘中1回
- 使用1ラウンド前に上下いずれかの行を表示
- 次のEnemy Phase、騎兵1体と弓兵1体がその行の伝令・護衛対象を優先
- 対象不在なら通常AIへ戻り、無料の追加攻撃は発生しない

#### 退路確保

- HP50%以下で一度だけ発動
- 次の行動終了までMOV+1、DEF-2
- 伝令所から4マスを超えて離れない

### 行動優先順位

1. 予告済み迂回命令を解決
2. HP50%以下で退路確保が未使用なら発動
3. 伝令箱へ到達可能な味方伝令を妨害
4. 通り抜け攻撃後に安全位置へ戻れる低HP対象を攻撃
5. 迂回命令が未使用なら対象行を予告
6. 伝令所から3マス以内で側面位置を取る

### 敵編成

- 高原運び手の隊長1
- 伝令騎兵1
- 槍兵2
- 弓兵2
- 先行伝令ルートでは3ラウンド目に軽装剣士1増援

### 報酬

| 条件 | Pending報酬・進行 |
|---|---|
| 主目的達成 | 獣皮2、織物2、騎具素材1 |
| 伝令所耐久8以上 | 騎兵運用記録 |
| 2人以上撤退・降伏 | 高原街道図を未取得なら追加 |
| 味方戦闘不能者0 | 騎具素材1 |
| 初回地域攻略 | 高原街道図・騎兵運用・標的指定・伝令路図の未取得分を追加 |

初回地域攻略時の未取得記録は、伝令所の街道台帳と保管箱から回収する。同じ記録は報酬台帳で
重複追加しない。各地点で先に取得した場合は地域攻略前の安全帰還から装備・スキル情報へ利用
できるため、副目標の早期達成には引き続き価値がある。

人間系の戦闘不能は死亡を意味しない。隊長は撤退後、集会所で街道共同運用の証言者として登場
できるが、仲間にはならない。

## 地域攻略と拠点接続

安全帰還時:

- `windscar_plateau_secured`を恒久登録
- 高原街道図で輸送情報と次地域の経路表示を解放
- 騎兵運用記録で機動訓練の上位候補と騎兵装備レシピを解放
- `plateau_targeting_records`で監視弓兵「標的指定」を解放可能
- `courier_route_chart`で辺境斥候「緊急離脱」を解放可能
- 騎具素材を伝令騎兵の武器分岐へ使用可能
- 旧辺境集落を次の遠征先へ追加

全UnitのMOVを恒久的に上げない。解放効果は装備、スキル、情報、護衛支援へ限定する。

- 斥候へ騎兵級の再移動を与えず、騎兵へ悪路無視を与えない
- 高MOV、再移動、射程を同じUnitへ過剰に集中させない
- 人間敵の戦闘不能を死亡として固定しない
- 輸送団を理由なく悪人化せず、街道維持の実績も示す
- 全UnitのMOVを恒久上昇させない
- 新兵種やレベル成長を追加しない

## 設定完了条件

- 6地点、地点3・4の攻略順選択、2キャンプ、4〜6周
- 強風、分散配置、護衛、荷車、伝令所の共通ルール
- 全地点の探索3択、任務、敵、報酬、敗北条件
- ボス数値、再移動、予告命令、OR主目的
- 最低保証、安定ID、周回短縮、安全帰還
- 機動訓練、騎兵装備、2スキル、旧辺境集落への接続

以後の敵数、風発生Round、報酬量は実戦テストに基づくバランス調整として扱う。
