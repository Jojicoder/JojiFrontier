# 第9地域 破砕された前線砦

状態: **JOJIFrontier地域仕様として確定**。7地点、複合任務、増援、指揮記録、施設最終分岐は
本書を正本とする。March Lineとエンバーマーチ軍制はWorld Bibleを正本とする。

内部地域IDは`shattered_march_fort`、表示名は「破砕された前線砦」。砦はFrontier project canon。

## 概要

聖堂の避難記録は、外縁施設を切り離してMarch Lineへ兵力を戻す命令が、この砦を経由したことを
示す。砦には撤退命令の断片と、命令に従わず残った守備隊の後継集団がいる。命令者を即座に裏切者
と断定せず、限られた兵力で内側を守る判断だった可能性を残す。

攻略とは**外郭、兵舎、兵站、信号庭、予備壁、指揮所、命令庫**を保全し、記録を持ち帰ること。

## 地点・周回

```text
1 破砕外郭 -> 2 崩れ門 -> CAMP I
                    ↓
              3 旧兵舎 / 4 兵站庫（順序選択） -> CAMP II
                    ↓
              5 信号庭 -> 6 予備壁 -> CAMP III
                    ↓
              7 切離命令庫
```

## 地形・Object

砦床35〜50%、瓦礫15〜25%、防壁床10〜15%(DEF+2)、射撃台5〜10%(射程攻撃の視界確保のみ)、
破孔5〜10%(増援口)、通行不能壁5〜10%。茂み・浅瀬はなし。

Object: 城門16、防護壁12、兵站箱6、信号盤8、警鐘6、指揮卓10、命令箱8。初期配置と必須Object
への経路を保証し、増援口をUnit初期配置へ重ねない。

## 敵勢力

- **残留砦隊**: 訓練済み。防壁、射撃台、兵站箱を守り、役割分担する
- **軍需回収団**: 物資を持って撤退。砦防衛はしない
- **残留砦隊長**: 命令記録の公開が旧守備隊を裏切者にすると考え、命令庫を封鎖する

## 最終強敵 残留砦隊長

HP48、STR10、DEF9、RES5、MOV4、射程1。固有行動:

- **交互防衛**: 戦闘中2回、上・下いずれかの増援行を1Round前に予告
- **壁際指揮**: 防壁隣接中、自身と最寄り味方2人DEF+2
- **記録封鎖**: 命令箱隣接中、味方は箱を操作できない
- HP25%以下かつ命令箱2個保全なら撤退。撃破は必須ではない

敗北優先、増援Batch、Object破壊、AND/OR Objectiveは`mission_objectives.md`に従う。
