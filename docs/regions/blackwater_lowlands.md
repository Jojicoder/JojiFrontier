内部地域IDは`blackwater_lowlands`、日本語表示名は「黒水低湿地」とする。湿地の動植物は
普通の生物を基準とし、毒や大型個体を魔法汚染の結果にしない。

## 地域概要

灰鉄採石場の先には、灰を含む水が低地へ溜まった広い湿地がある。水は黒く見えるが、呪いや
魔力汚染ではない。灰、泥、腐植、鉄分が混じり、浅い場所と踏み抜く深泥を見分けにくい。

旧街道は水路と植生に覆われ、薬草採取者と樹脂回収者が細い足場を利用している。遠征隊は彼らを
追い払うのではなく、安全な渡り道と採取量の上限を決める必要がある。毒を持つ蜘蛛や蛇、縄張りを
持つ大型個体は戦闘上の脅威になる。

## 開拓できない理由

- 安全な浅瀬と深泥の区別が記録されていない
- 毒を持つ生物と古い罠が街道予定地へ集中している
- 薬草と湿地樹脂の乱採取で現地利用者との衝突が起きている
- 沈没した水門が水位を上げ、同じ道を次回も使える保証がない
- 深部の大型蛇が水源と乾いた足場を縄張りにしている

地域攻略は、**安全路、採取規則、治療手順、水位管理、水源周辺の通行**を確立することを意味する。

## 地域の役割

- 遠征をまたぐHPとアイテム消耗を本格的に検証する
- 毒、状態治療、万能薬、浅瀬解除を実戦へ接続する
- 全12兵種から4人を選ぶ価値を探索と戦闘の両方で示す
- 資源を多く採るほど敵増援・毒・帰還判断が増える構造にする
- 診療所「薬学」と工房「罠技術」を解放する
- 新兵種や恒久能力値上昇は追加しない

## 湿地の共通ルール

### 毒

- 毒攻撃が命中、または毒溜まりで行動終了すると確定付与
- 対象側Phase終了時2ダメージ、3回、HP1で止まる
- 万能薬または暁の衛生兵「状態治療」で解除
- 戦闘終了時に解除し、次戦闘へ持ち越さない

### 採取

- 採取地点で専用行動を選ぶと、そのユニットの行動を終了する
- 敵が残っていても採取可能
- 採取報酬は勝利後にPendingへ追加し、途中撤退では得られない
- 同じ採取地点から同一戦闘中に複数回取得できない
- 採取数を増やす選択には増援、毒、時間制限のいずれかを必ず伴わせる

## 敵勢力

### 湿地の毒蜘蛛

普通の蜘蛛が稀に大きく育った地域個体。群れで獲物を囲むが、人間の装置や任務目標は理解しない。

- 毒を持たない対象を優先して攻撃
- 茂みと深泥を経路として評価する
- 2体以上で同じ対象へ接近可能なら孤立対象を包囲
- HPは低め、MOVは高め、DEFは低い

### 沼蛇

- 浅瀬から隣接攻撃できる位置を優先
- 毒状態の対象には通常攻撃、未毒対象には毒噛みを優先
- 単独行動し、蜘蛛の包囲AIを共有しない

### 樹脂回収者

現地資源を利用する人間集団。全員を敵にせず、過剰採取や罠の所有を巡って対立した班だけが戦う。

- 鉱石ではなく樹脂箱と脱出路を優先
- HP30%以下で撤退を評価
- 降伏・撤退した人物を死亡扱いにしない
- 主な兵種: 斧兵、弓兵、罠師

## 地域ボス 沼牙の大蛇

魔物ではなく、湿地で長く生きた大型個体。人間を追って地域外へ移動せず、水源と乾いた足場を
守る縄張り行動を取る。

### 基本値

| 項目 | 値 |
|---|---:|
| HP | 60 |
| STR | 9 |
| MAG | 0 |
| DEF | 6 |
| RES | 3 |
| MOV | 4 |

### 行動

#### 毒牙

- 射程1、STR+3物理ダメージ
- 命中時に毒を確定付与
- すでに毒状態なら追加ダメージを増やさない

#### 水中潜行

- 1ラウンド前に移動先候補2マスを表示
- 次のEnemy Phase開始時、どちらか一方へ移動して隣接1体を攻撃
- 辺境猟兵ルートでは実際の移動先1マスだけを表示
- 浅瀬が経路にない場合は使用しない

#### 締め付け

- 隣接する味方が2体以上いる場合に前方3マスを攻撃
- STR+1物理ダメージと移動低下
- 同じ対象へ効果量を重複させない

#### 激しい身震い

- HP50%以下で一度だけ使用
- 隣接4マスのユニットを1マス押し出す
- 毒溜まりへ押し出された場合は行動終了時に毒付与
- 水源標識は押し出さず、固定2ダメージを受ける

### 行動優先順位

1. 予告済み水中潜行を実行
2. HP50%以下で身震いが未使用なら使用
3. 水源標識を確保中の味方を攻撃
4. 前方3マスに味方2体以上なら締め付け
5. 未毒の隣接対象へ毒牙
6. 浅瀬を使って孤立対象へ潜行予告
7. それ以外は水源から3マス以内を維持

### ボス補正

- 毒: 1ダメージ、最大2回
- 炎上: 2ダメージ、最大2回
- 移動低下: MOV-1
- 防御低下: DEF-2
- よろめき: 次の行動だけMOV-1
- 完全な行動不能は無効

### 報酬

| 条件 | Pending報酬・進行 |
|---|---|
| 主目的達成 | 薬草2、高品質薬草1、毒素材2、湿地樹脂1 |
| 標識2個確保 | 湿地踏査記録を未取得なら追加 |
| 毒状態0 | `marsh_emergency_medicine`を未取得なら医療箱から追加 |
| 薬草地点未使用 | 高品質薬草1 |
| 初回地域攻略 | 湿地踏査・薬学・罠技術・緊急処置の未取得記録を追加 |

初回地域攻略時の未取得記録は、水門の保管箱と水源調査結果から回収する。同じ記録は報酬台帳で
重複追加しない。各地点で先に取得した場合は地域攻略前の安全帰還から研究やスキル解放へ利用
できるため、副目標を早期に達成する価値は残る。

## 地域攻略と拠点接続

安全帰還時:

- `blackwater_lowlands_secured`を恒久登録
- 診療所「薬学」を研究可能にし、万能薬を解放候補へ追加
- 工房「罠技術」を研究可能にし、鉄杭と毒罠処理道具を解放候補へ追加
- `marsh_emergency_medicine`で暁の衛生兵「緊急処置」を解放可能
- 湿地の薬草、毒素材、樹脂を通常素材の再採取先へ追加
- 風裂き高原を遠征先へ追加

施設ノードは自動建設しない。重要発見と素材を持ち帰り、対応施設が稼働中の時だけ研究・製作を
行う。次地域解放は施設の稼働状況に依存させない。

## 実装順

1. 浅瀬、深泥、茂み、薬草地点、毒溜まり
2. 毒付与、万能薬、状態治療、AI評価
3. 7地点進行、3キャンプ、地点3・4の攻略順選択
4. 採取、護衛、脱出、水門Object
5. 各地点の敵編成、増援、報酬
6. 沼牙の大蛇、潜行予告、水源標識
7. 周回短縮、最低保証、重要発見、施設接続

## 実装時の不変条件

- 黒い水を魔法汚染や呪いにしない
- 野生動物をすべて魔物として扱わない
- 毒の付与確率を導入せず、命中時は確定付与
- 毒だけでHP0にしない
- 毒溜まりを初期配置や必須Objectiveマスへ生成しない
- 万能薬または衛生兵なしでも通常ルートを攻略可能にする
- 薬草地点を無限回復地点にしない
- 現地利用者を全員敵や略奪者として扱わない
- 新兵種、レベル、一律能力値上昇を追加しない

## 設定完了条件

- 7地点、地点3・4の攻略順選択、3キャンプ、4〜6周
- 湿地地形、毒、採取、護衛、水門の共通ルール
- 全地点の探索3択、任務、報酬、敗北条件
- ボス数値、予告行動、状態異常補正
- 最低保証、安定ID、周回短縮、安全帰還
- 薬学、罠技術、緊急処置、風裂き高原への接続

以後の敵数、生成率、報酬量は実戦テストに基づくバランス調整として扱う。
