# 第8地域 埋没聖堂

状態: **JOJIFrontier地域仕様として確定**。6地点、聖域、医療記録、遺跡装置、報酬、拠点接続は
本書を正本とする。暁の教会、Sanctuary、回復魔法、技術上限はWorld Bibleを正本とする。

内部地域IDは`buried_dawn_sanctum`、表示名は「埋没聖堂」。聖堂はJOJIFrontier project canon。

参照: World Bible `religion.md`、`docs/world/magic.md`、`docs/world/technology.md`、
`docs/world/embermarch.md`、Frontier `campaign_regions.md`、`base_development.md`、
`mission_objectives.md`、`status_effects.md`。

## 概要と境界

燼火峡谷の観測記録が示した先には、崩土に埋もれた暁の教会の救護聖堂がある。ここは戦勝を祈る
神殿ではなく、負傷者の治療、避難、記録保存を担った施設だった。内部には保護を求める回収者も
おり、Sanctuaryの慣習により、聖堂内から力ずくで連れ出してはならない。

- 医療典籍は蘇生、永久欠損の即時治療、無限回復を可能にしない
- 聖堂器具と魔法記録は希少な専門技術で、一般兵へ魔法を普及させない
- 回収者の犯罪容疑は消えないが、聖域内では交渉を優先する
- 灰嵐以前の記録は原因を断定せず、避難・負傷・物資移送だけを確定する

## 開拓できない理由

参道崩落、負傷者の残留、写本散逸、封鎖扉の故障、聖域を巡る対立がある。攻略とは
**参道、救護室、写本庫、封鎖回廊、祭壇記録を保全して避難路を作ること**である。

## 地点と周回

```text
1 埋没参道 -> 2 崩れた礼拝堂 -> CAMP I
                    ↓
              3 救護室 / 4 写本庫（順序選択） -> CAMP II
                    ↓
              5 封鎖回廊 -> 6 夜明け祭壇
```

標準5周、早期4周、回収6周。遠征は常に地点1から開始する。

| 成果 | 次回以降の短縮 |
|---|---|
| 参道補強 | 地点1を安全通過 |
| 礼拝堂退避所 | CAMP Iで状態異常を全解除。HP自動回復なし |
| 救護室復旧 | CAMP IIで救急セット1個補充、1遠征1回 |
| 写本目録 | 未取得記録とObjective位置を全公開 |
| 回廊開通 | 地点5を安全通過 |
| 聖堂保全 | 地域攻略、再訪は医療・記録回収のみ |

Object: 救護台耐久8、写本箱5、封鎖輪10、避難扉12、祭壇記録箱8。必須Objectと初期配置を
封鎖せず、最低1本の通路を保証する。

## 敵勢力

- **聖堂回収団**: 写本箱と器具を持って撤退。HP30%以下で降伏・撤退を評価
- **崩土の野生獣**: 普通の蛇・蜘蛛。人間用装置を理解せず孤立者を狙う
- **聖堂回収団長**: 強敵だが最終攻略に撃破不要。聖域内の人物へ攻撃しない

- 地点1: 参道支柱2本保全 -> 建築材+1
- 地点2: 避難者全員脱出 -> 野戦救護記録
- 地点3: 救護台耐久6以上 -> 医療典籍

## 最終強敵 聖堂回収団長

HP44、STR9、DEF7、RES6、MOV4。射程1。写本箱へ近づくUnitを優先し、戦闘中1回、次の増援行を
予告する「退去命令」を使う。HP25%以下、記録箱2個保全、退路ありなら降伏する。撃破・降伏の
どちらでも主目的を阻害しない。

## 地域攻略と最低保証

安全帰還時:

- `buried_dawn_sanctum_secured`
- 開拓都市への発展候補
- 特殊医療、魔導工房、聖堂装置研究
- 埋没聖堂の証言と灰嵐以前の医療記録
- 破砕された前線砦を解放

最低保証: 建築材6、石材4、高品質鉄材2、薬草4、聖堂器材3、遺跡片4、医療典籍1、
上位魔法研究記録1、聖堂装置記録1。開拓都市化は拠点画面で建築材6、石材4、高品質鉄材2を
消費して確定し、重要発見は消費しない。

## 不変条件

- Sanctuaryを無罪判定や強制連行に変えない
- 蘇生、無限回復、魔法一般化を導入しない
- 強敵撃破を必須にしない
- 医療記録で灰嵐の原因を断定しない
- 全24マスを生成し、必須Objectiveへの経路を保証する
- 内部ID、英語名、Seedを通常画面へ表示しない

## 実装受入条件

- 6地点、2キャンプ、4〜6周が成立する
- Sanctuary交渉、避難、写本運搬、装置操作を敵全滅以外で達成できる
- 医療典籍と必須素材を探索選択にかかわらず取得可能
- 強敵未撃破でも聖堂保全と安全帰還で地域攻略できる
- 開拓都市、特殊医療、魔導工房、次地域を重複なく解放する
