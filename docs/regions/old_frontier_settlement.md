## 地域概要

風裂き高原の伝令路を越えると、かつてMarch Line外縁の補給を担った集落が残っている。完全な
廃墟ではなく、離れずに暮らしてきた残留住民、避難先から戻った帰還者、空いた建物を利用する
運搬人がいる。

遠征隊が街道復旧を理由に土地と建物を一方的に接収すれば、開拓は新しい追放になる。一方で、
井戸、穀物庫、防壁を共同管理しなければ、襲撃と物資不足で集落は維持できない。遠征隊は所有権を
裁定するのではなく、王権による正式確認までの暫定利用協定と共同防衛体制を作る。

## 法的境界

- 遠征隊は土地の恒久所有権、相続権、徴税権を付与しない
- 空き家を「無主」と断定せず、台帳と証言を照合する
- 残留住民、帰還者、運搬人の利用実績を記録する
- 重大犯罪の裁定や爵位付与を行わない
- 地域攻略後の協定は暫定であり、エンバーマーチ側の正式承認を必要とする
- 戦闘で降伏・撤退した人間を処刑せず、拘束または退去として扱う

## 開拓できない理由

- 井戸と穀物庫の利用順が決まらず、集団同士が対立している
- 古い住民台帳が散逸し、建物の利用根拠を確認できない
- 防壁と避難所が壊れ、襲撃時に住民を守れない
- 食料、建築材、鉄材を誰が管理するか決まっていない
- 街道を狙う襲撃団に対し、各集団が別々に防衛している

地域攻略は、**水、食料、住居、記録、防衛**について暫定合意を作り、共同防衛を成功させることを
意味する。

## 地域の役割

- 敵全滅以外の交渉結果と中立Unitを扱う
- 住民、物資、避難所を同時に守る複合防衛を行う
- 行軍隊長、古参守備兵、旗手、工兵の支援差を検証する
- 共同施設研究と交流区画を解放する
- `settlement_command_ledger`と`collective_defense_records`を取得する
- 新兵種、レベル、恒久能力値上昇は追加しない

## 地点構成

```text
1. 風化した外柵
        ↓
2. 共同井戸
        ↓
    キャンプI
        ↓
3. 旧穀物庫 ─────┐
                  ├─ 攻略順を選択
4. 集会家屋 ─────┘
        ↓
    キャンプII
        ↓
5. 夜明けの共同防衛
        ↓
地域攻略・燼火峡谷解放
```

- 地点1・2は順番固定
- 地点3・4はどちらを先に進めてもよい
- 片方の成果だけを安全帰還で恒久化できる
- 最終防衛には地点3・4の両方が必要
- キャンプは地点2と4の進行段階後
- 遠征は常に地点1から開始する

## 周回と短縮

標準4周、副目標回収を含め5周を想定する。

| 恒久成果 | 次回以降の効果 |
|---|---|
| 外柵通路 | 地点1を安全通過 |
| 井戸利用協定 | キャンプIで生存者全員HP2回復。1遠征1回 |
| 穀物庫共同管理 | キャンプIIで野営食1個を補充。1遠征1回 |
| 集落台帳復元 | 最終防衛の中立住民配置と敵増援を全公開 |
| 共同防衛協定 | 地域攻略。再訪は防衛訓練、会話、素材回収のみ |

## 地形生成

全戦闘は3行×8列。24マスすべてを遠征Seedから生成する。

| 地形 | 生成目安 | 効果 |
|---|---:|---|
| 集落道 | 35〜50% | 移動コスト1 |
| 荒れた庭 | 10〜20% | 移動コスト2。辺境斥候は1 |
| 低い石垣 | 10〜15% | 移動コスト1、DEF+2 |
| 家屋跡 | 10〜20% | 移動コスト2 |
| 共同広場 | 5〜10% | 移動コスト1。任務地点候補 |

## 集団と戦術

### 残留住民

集落を離れず、井戸と建物を維持してきた人々。戦闘では中立扱いで、武装していても遠征隊の
指揮下Unitにはしない。

### 帰還者

避難先から戻り、古い居住記録を根拠に家屋と畑の利用を求める。残留住民と対立するが、最初から
敵対勢力にはしない。

### 灰道襲撃団

街道輸送と集落備蓄を狙う武装集団。

- 食料箱、井戸、避難所の順に戦術目標を評価
- 斧兵は外柵と防護柵を破壊
- 弓兵は住民と警鐘への射線を取る
- 軽装兵は物資を確保して敵側退路へ撤退
- HP30%以下で撤退を評価し、戦闘不能を死亡扱いにしない

### 探索3択

| 選択 | 条件 | 効果 |
|---|---|---|
| 台帳と証言を照合する | なし | 記録箱3個、敵4体 |

### 任務と報酬

- 主目的: 記録箱2個以上を集会家屋へ運ぶ
- OR条件: 敵全滅後、残る記録箱1個を操作
- 副目標: 3個すべて回収

資料として扱う。

### キャンプII

穀物庫共同管理と集落台帳を安全帰還で確定すると利用可能。最終防衛の住民配置、備蓄、防護Object
が事前表示される。

## 5. 夜明けの共同防衛

襲撃団が備蓄と井戸を狙って三方向から接近する。遠征隊は住民を指揮下兵士に変えるのではなく、
避難、警鐘、物資搬送を分担しながら集落を守る。

### 探索3択

| 選択 | 条件 | 効果 |
|---|---|---|
| 外柵を中心に守る | なし | 防護柵2個、上段増援が多い |
| 住民を先に避難させる | なし | 中立Unitなし、避難所耐久-3、敵1体追加 |
| `[旗手]` 防衛班を三つに分ける | 旗手 | 3組分散配置、警鐘を開始時に1個操作済み |

### 主目的

次のAND条件をすべて達成する。

1. 5ラウンド終了まで避難所を守る
2. 共同井戸または穀物庫のどちらかを耐久1以上で残す
3. 警鐘を1回以上操作する

副目標:

- 井戸と穀物庫を両方保全
- 中立住民を全員避難
- 襲撃団頭領を撤退させる
- 味方戦闘不能者0

敗北条件:

- 部隊全滅
- 避難所耐久0
- 中立住民全員が撤退し、避難完了者も0

### 敵編成と増援

- 初期: 襲撃団頭領1、斧兵2、弓兵2
- 2ラウンド目: 軽装剣士2、出現位置を1ラウンド前に表示
- 4ラウンド目: 斧兵1、弓兵1、出現位置を1ラウンド前に表示
- 警鐘操作で次の増援位置と兵種を完全表示
- 襲撃団頭領が撤退しても残る敵は即消滅せず、通常の撤退評価へ移る

## 強敵 襲撃団頭領

この地域の最終目的は頭領撃破ではなく共同防衛。頭領は強敵だが、無視して5ラウンド守ることも
できる。

| 項目 | 値 |
|---|---:|
| HP | 42 |
| STR | 10 |
| MAG | 0 |
| DEF | 7 |
| RES | 3 |
| MOV | 4 |
| 武器 | 重斧、威力7、射程1 |

固有行動:

- **柵割り**: Objectへ与えるダメージ+4。Unitへの追加補正なし
- **略奪指示**: 戦闘中1回、次の増援が穀物庫を優先することを1ラウンド前に予告
- **退路判断**: HP30%以下かつ4ラウンド目以降なら敵側退路へ撤退

行動優先順位:

1. 予告済み略奪指示を解決
2. 避難所へ通じる防護柵を破壊
3. 穀物庫または井戸へ到達可能ならObjectを攻撃
4. 自身を戦闘不能にできる味方を攻撃
5. HP30%以下なら撤退

### 最終報酬

| 条件 | Pending報酬・進行 |
|---|---|
| 主目的達成 | 建築材2、鉄材2、食料2 |
| 井戸・穀物庫保全 | 集団防衛資料、`collective_defense_records` |
| 全住民避難 | 織物2 |
| 頭領撤退 | 鉄材1 |
| 味方戦闘不能者0 | 建築材1 |
| 初回地域攻略 | 集落台帳・集団防衛資料の未取得分を追加 |

重要発見の未取得分は、防衛後に残った集会家屋の記録と住民証言から回収する。重複追加しない。

## 地域攻略と拠点接続

安全帰還時:

- `old_frontier_settlement_secured`を恒久登録
- 共同施設研究`collective_facility_methods`を解放
- 宿舎「交流区画」を研究可能にする
- 集会所へ残留住民、帰還者、運搬人、防衛参加者の会話を追加
- `settlement_command_ledger`で行軍隊長「援護命令」を解放可能
- `collective_defense_records`で古参守備兵「不動の構え」を解放可能
- 燼火峡谷を遠征先へ追加

共同施設研究は施設数を増やす効果ではなく、診療所・訓練所・工房・鍛冶場の地域共同作業を
研究可能にする。建設済み施設は以前から常時利用できる。

## 実装時の不変条件

- 遠征隊が土地の恒久所有権や徴税権を勝手に与えない
- 残留住民と帰還者の一方だけを正義・悪として固定しない
- 全住民を戦闘Unitや加入候補にしない
- 強敵撃破を最終防衛の必須条件にしない
- 人間の戦闘不能を死亡として固定しない
- 共同施設研究を能力値の一律上昇へ変えない
- 任意会話の未読で主進行を停止しない
- 新兵種、レベル、恒久能力値上昇を追加しない
