内部地域IDは`ember_ravine`、日本語表示名は「燼火峡谷」とする。峡谷の熱は地熱、噴気、燃える
天然油、乾燥植生によるもので、魔法汚染ではない。硫黄と燃焼油は火薬へ加工できず、銃砲・爆薬を
導入しない。

## 地域概要

旧辺境集落から南東へ続く峡谷には、熱を利用した旧加工所と観測所が残る。噴気孔と天然油の火が
道を分断し、冷却水路は壊れている。採取者は耐熱素材、硫黄、灰晶を求めるが、採取を急ぐほど
炎上範囲と崩落が広がる。

観測所には、灰嵐以前にも外縁監視所が複数回沈黙した記録がある。ただし原因は欠落しており、
自然災害、獣害、人間同士の争い、外側の存在のいずれとも断定しない。

## 開拓できない理由

- 噴気と炎上床が街道を周期的に塞ぐ
- 冷却水路と遮熱扉が壊れ、安全な退避場所がない
- 採取班が別々に火を扱い、峡谷全体の熱量を管理していない
- 旧加工所の耐熱技術が失われ、素材を安全に運べない
- 最深部の大型蜥蜴が冷却水場と観測路を縄張りにしている

地域攻略は、**安全路、冷却、水門、採取規則、加工所、観測所、深部封鎖**を一体で復旧することを
意味する。

## 地域の役割

- 炎上床と浅瀬・冷却床による解除判断を本格化する
- 戦場全体の熱量を公開情報として管理する
- 危険な採取と安全な環境制御を比較する
- 工兵、重装兵、衛生兵、戦闘魔導士の異なる対処を使う
- 特殊鍛造、耐熱加工、上位戦闘工作を解放する
- 灰嵐以前の記録を埋没聖堂へ接続する

## 共通地形

全戦闘は3行×8列。24マスすべてを遠征Seedから生成する。

| 地形 | 生成目安 | 効果 |
|---|---:|---|
| 焼け石床 | 30〜45% | 移動コスト1 |
| 熱砂 | 10〜20% | 移動コスト2 |
| 炎上床 | 10〜20% | 行動終了時に炎上を確定付与 |
| 冷却床 | 5〜15% | 移動コスト1。行動終了時、炎上ダメージ前に炎上解除 |
| 噴気予告床 | 5〜15% | 次のRound終了時に炎上床化。1ラウンド前に表示 |
| 灰煙床 | 5〜10% | 移動コスト2。回避補正なし |
| 岩壁・遮熱扉 | 5〜10% | 通行不可または耐久Object |

- 炎上床、噴気床を初期配置、脱出、必須Objectiveマスへ生成しない
- 3行すべてを炎上床と障害物で封鎖しない
- 冷却床へ到達できる経路を最低1本保証する
- 初期配置Unitのマスだけ必要時に焼け石床へ変更する
- 灰煙床は命中率を下げない。煙幕筒のマス効果とは別物

## 戦場熱量

熱量は戦闘ごとの共有値で`0〜3`。戦闘前とHUDへ常時表示する。

| 熱量 | 効果 |
|---:|---|
| 0 | 追加効果なし |
| 1 | 噴気予告を通常処理 |
| 2 | 各Round開始時、空きマス1個を噴気予告床にする |
| 3 | 各陣営Phase終了時、冷却床以外の全Unitへ固定1ダメージ。HP1で止まる |

- 熱量上昇と低下は装置操作または探索結果だけで発生し、ランダム変動しない
- 噴気弁破壊で+1、冷却弁操作で-1
- 1回の操作で2段階以上変化させない
- 熱量ダメージは炎上状態ではなく、万能薬で解除できない
- 戦闘終了時に熱量を破棄し、次戦闘へ直接持ち越さない

## 敵勢力

### 熱地採取団

耐熱素材と硫黄を回収する人間集団。全員を敵とせず、採取権と水路操作を巡って武装衝突する班だけ
戦闘へ参加する。

- 素材箱、冷却弁、脱出路を優先
- 炎上中は到達可能な冷却床を評価
- HP30%以下で撤退
- 主な兵種: 斧兵、弓兵、工兵型

### 岩蜥蜴

峡谷の熱い岩場に適応した普通の爬虫類。大型個体は危険だが魔物ではない。

- 冷却床を避け、熱砂と焼け石床を優先
- 炎上無効ではなく、各戦闘最初の炎上付与だけ無効
- 孤立対象と装置操作者を優先
- 人間のObjectiveや素材箱は理解しない

## 地域ボス 赤背の大蜥蜴

魔物ではなく、地熱地帯で大型化した縄張り個体。

| 項目 | 値 |
|---|---:|
| HP | 64 |
| STR | 10 |
| MAG | 0 |
| DEF | 8 |
| RES | 3 |
| MOV | 4 |

行動:

- **熱砂突進**: 1ラウンド前に直線3マスを予告。通過UnitへSTR+3物理ダメージ
- **尾払い**: 前上・正面・前下へSTR+1物理ダメージと1マス押し出し
- **噴気誘導**: HP50%以下で一度、空き2マスを噴気予告床化
- **冷却回避**: 冷却床を経路評価-3とし、他経路がなければ通行可能

行動優先順位:

1. 予告済み突進
2. HP50%以下で噴気誘導
3. 冷却弁の操作者を攻撃
4. 前方3マスに味方2体以上なら尾払い
5. 突進可能な孤立対象へ予告
6. 封鎖扉から3マス以内を維持

通常のボス状態異常補正を適用し、完全行動不能は無効。

## 地域攻略と拠点接続

安全帰還時:

- `ember_ravine_secured`を恒久登録
- 鍛冶場「特殊鍛造」「耐熱加工」を研究可能にする
- 工房「上位戦闘工作」を研究可能にする
- 制御燃焼式で戦闘魔導士の残火系装備候補を解放
- 耐熱素材、硫黄、灰晶を製作素材へ追加
- 灰嵐以前の監視記録を集会所へ追加
- 埋没聖堂を次の遠征先へ追加

硫黄と燃焼油から火薬・爆薬・銃器は製作できない。用途は薬品、保存、着火、熱加工、火炎壺に
限定する。

- 熱量、噴気、炎上床を事前表示する
- 冷却手段なしで必須ルートを炎上床封鎖しない
- 炎上を隣接へ自動拡散させない
- 戦闘魔導士だけを必須解法にしない
- 監視記録から外側の正体を断定しない
- 新兵種、レベル、恒久能力値上昇を追加しない

## 設定完了条件

- 8地点、3キャンプ、4〜6周
- 熱量、炎上、冷却、噴気、採取の共通ルール
- 全地点の探索3択、任務、報酬、敗北条件
- ボス数値と予告行動
- 最低保証、安定ID、周回短縮、安全帰還
- 特殊鍛造、上位戦闘工作、監視記録、埋没聖堂への接続

以後の熱量、敵数、生成率、報酬量は実戦テストに基づくバランス調整として扱う。
