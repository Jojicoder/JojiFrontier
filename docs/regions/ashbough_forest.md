# 第1地域: 灰枝の森

## 状態

JOJIFrontierの第1地域として採用を確定したシナリオ案。地域名・3地点名・ボス名はローカルに
確定済みで、実装はこの名称で進めてよい。地域ゲーム仕様も初期実装値まで確定済みとする。
World Bible側への正式命名申請は実装をブロックしない
別タスクとして扱う（承認後に表記差分があれば追随修正する）。

`implementation_roadmap.md`のPhase 2範囲(灰枝の林縁のみ)を実装済み。`jf::RegionId`/
`StageDescriptor`によりCinderwatch Gateと並ぶ2つ目の地域としてデータ駆動で追加した
(`jf/core/Region.hpp`)。実装済み: 狼4体(急行ルートのみ3体)、群れAI(`jf/battle/EnemyAI.cpp`の
`takeWolfPackTurn`)、探索3択(通常・急行・斥候ルート)とルート別報酬、踏査地点(SecureTile副目標)
のランダム有効配置と成功時の木材+1ボーナス、`GameApp::startExpedition(RegionId::AshboughForest)`
経由でのフル遠征(開始→戦闘→キャンプ→安全帰還)。

実装状況の詳細は[`../implementation_status.md`](../implementation_status.md)を正本とする。本書では

### 正史上の所属

- 大陸: Asteria
- 国: Embermarch王国
- マクロ地域: South / Southern Ashlands
- 前線: Embermarch南端のMarch Line周辺

EmbermarchはHouse Bloodmarchが統治するAggressive / Military国家で、南方の未踏領域に
対してMarch Lineを維持している。JOJIFrontierのプレイヤーはWarmarchそのものではなく、
前線拠点を運営するExpedition Marshalである。

### 正史との整合

World Bibleの`flora_fauna.md`では、普通の捕食動物、縄張りを持つ動物、追いやられた群れ、
稀な大型個体が街道や辺境開拓地で戦闘上の脅威になり得る。これは魔法生物の常設ベスティアリや
国家規模の怪物狩り組織を意味しない。

灰角大猪はこのルールに沿う稀な大型個体とする。魔物や未知の固有種ではなく、火災、乾燥、
軍の往来、新規開拓による生息域の変化で人間と衝突した普通の猪である。March Line外に何が
存在するかという意図的な空白は変更しない。

## 地域の目的

拠点の先に広がる森林で大型獣の活動により街道予定地が使えない。

- 調査隊が襲われている
- 木材や薬草を採取できない
- 安全な水場を確保できない
- 開拓民を送り込めない

目的は森の征服ではなく、安全な通行路の確保である。

## 地域構造

最初の短い導入遠征として3地点で構成する。

1. 灰枝の林縁
2. 薬草の沢
3. 折れ木の縄張り

地点ごとの役割を情報、資源、ボス条件に分け、同じ判断を繰り返さない。

### 地域共通の勝敗条件

- 通常戦の主目的は指定された敵と未解決の必須増援をすべて戦闘不能にすること。ただし薬草の沢は
  薬草確保後の撤収をOR条件として許可する
- ボス戦の主目的は灰角大猪のHPを0にし、任務上の撤退を成立させること
- 全戦闘の敗北条件は出撃した味方全員の戦闘不能
- 主人公、行軍隊長、特定兵種の戦闘不能だけでは敗北しない
- ラウンド制限、護衛対象、防衛地点、破壊禁止Objectは灰枝の森では使用しない
- 手動撤退は敗北扱いとし、その遠征のPending資産と未確定地域進行を失う
- 同じEvent Batchで主目的と全滅が成立した場合は敗北を優先する
- 副目標の未達成または失敗だけでは戦闘を終了しない

### 想定周回数

灰枝の森は3地点だけの導入地域なので、本格地域の4〜6周基準を適用しない。

- 標準攻略は2周
- 第3周は取り逃した踏査、薬草、高品質薬草などの任意回収
- 第1周でボスまで到達できる場合は制限せず、そのまま攻略可能
- 周回数による強制ロックは設けない

想定進行:

1. 第1周: 林縁と薬草の沢を踏査し、キャンプから安全帰還
2. 第2周: 確保済みの2地点を安全通過し、折れ木の縄張りと灰角大猪を攻略
3. 第3周: 必要な場合だけ未達成副目標と追加素材を回収

この表は灰枝の森を実装する際の確定基準値とする。地域自体は設計済み・未実装である。

| 地点 | 基本勝利報酬 | 選択・副目標による追加／減少 | ランダム生成で保証する特殊タイル | 地形生成率 |

| 薬草の沢 | 木材1 | 採取ルートで薬草2、共通薬草地点で薬草+1、衛生兵ルートで高品質薬草1 | 中央4列に浅瀬と薬草地点2マス、衛生兵ルートでは専用調査地点 | 通常床40%、浅瀬35%、茂み15%、灰地10% |

通常素材とは別にObjectiveとPending Discoveryを得る。灰角の大牙はDiscoveryではなく、
安全帰還後に拠点発展へ使用するキー素材である。

### 戦闘マップ生成共通ルール

灰枝の森の全戦闘マップは固定レイアウトを持たず、戦闘ごとに3x8盤面をランダム生成する。

- 遠征開始時のseed、地域ID、地点ID、戦闘番号から地点ごとのseedを作る
- 新しい遠征では異なる盤面になる
- セーブ読込や同じ遠征の再開では同じ盤面を復元する
- 全24マスを先に生成し、実際に初期配置されたユニットの足元が通行不可の場合だけ通常床へ変える
- 出撃範囲内の未使用マスは、障害物を含む生成済み地形をそのまま維持する
- 味方側から敵側まで最低1本の通行可能経路を保証する
- Barrier、倒木、浅瀬などが3行すべてを同じ列で封鎖しない
- Objective、薬草、倒木を通常地形生成後に配置し、到達可能性を再検証する
- 条件を満たせない生成結果は採用せず、同じseedから決定論的に再生成する
- 探索ルートの効果は固定マス指定ではなく、生成条件と配置数の変更として反映する

「常にランダム」は画面を開き直すたびに再抽選する意味ではない。戦闘開始後のリロードで有利な
盤面を引き直せないよう、開始済み戦闘の地形と特殊タイルは保存対象にする。

### 地形生成率の適用

- 表の比率は特殊ObjectとObjectiveを置く前の24マスへ使用する重み
- 24マスで各比率の厳密な個数を保証せず、Seed付き重み抽選とする
- 薬草の沢は浅瀬を最低5マス、最大8マスに収める
- 浅瀬は最低3マスが上下左右で連結するよう再配置する
- 林縁の茂みは最低2マス、最大4マス。縄張りは最低2マス、最大7マス
- 味方と敵の初期配置を開く処理後、特殊地形数が最低数を下回っても再追加しない
- 経路、Objective、初期配置の成立条件を地形比率より優先する
- 生成再試行は同一Seedから派生番号を増やして最大32回。失敗時は決定論的な通常床配置へ縮退する

## 地点進行と安全通過

地点状態は[`../exploration_system.md`](../exploration_system.md#周回と地域経路の開拓)の
未踏、踏査済み、経路確保済みを使用する。

| 地点 | 踏査済み条件 | 経路確保済み条件 | 次回以降の安全通過 |
|---|---|---|---|
| 灰枝の林縁 | 戦闘勝利 | 踏査地点を確保して安全帰還 | 狼戦と探索3択を省略。報酬なし |
| 薬草の沢 | 戦闘勝利 | 衛生兵専用踏査を達成して安全帰還 | 狼戦を省略。薬草採取なし |
| 折れ木の縄張り | ボス戦開始 | 灰角大猪を退かせて安全帰還 | 地域攻略後の再訪時だけ使用。ボス再戦なし |

- 踏査済みでは敵編成、特殊地形数、探索リスクを事前表示する
- 経路確保済みでは「安全路を進む」と「危険区域を再調査」を選択できる
- 再調査は新しい盤面で通常戦闘を行い、通常素材だけ再取得できる
- 初回Objective、重要発見、灰角の大牙、加入条件は再取得できない
- 林縁だけ確保済みの場合、林縁を安全通過して薬草の沢から通常進行する
- 薬草の沢の経路確保に暁の衛生兵を必須とするが、地域攻略自体には必須としない
- 衛生兵なしで攻略した場合、第3周以降に衛生兵を編成して沢を確保できる

### 特殊タイル効果正式表

| タイル | 通行 | 移動コスト | 防御補正 | 行動・戦闘効果 | 消滅・取得条件 |
|---|---|---:|---:|---|---|
| 倒木 | 通行不可 | - | - | 灰角大猪の突進を停止する。衝突時、大猪は次のEnemy Phaseを失い、スタン中はDEF 2・RES 0になる | 突進衝突時に破壊。A/Cで未破壊のまま勝利すると木材+1 |
| 薬草地点 | 通行・停止可能 | 1 | 0 | 味方がこのマスで行動終了するとHP5回復し、採取済みになる | 採取後は通常床へ変化。採取済み状態で勝利すると薬草+1。戦闘ごとに1回のみ |
| 浅瀬 | 通行・停止可能 | 2 | 0 | 地上ユニットの移動を遅らせる。攻撃、回避、ダメージへの補正はない | 戦闘中は残り続ける |
| 茂み | 通行・停止可能 | 1 | 0 | 上にいるユニットへの攻撃命中率を20%下げる（通常100%から80%） | 戦闘中は残り続ける |

追加ルール:

- 倒木は通常攻撃、回復、アイテムの対象にできない
- 倒木を破壊できるのは現段階では灰角大猪の突進だけ
- 薬草地点で行動終了すると、そのユニットの行動は通常どおり終了する
- 薬草地点の回復は最大HPを超えず、HP満タンでも採取と行動終了は発生する
- 2つの薬草地点は到達ルートの選択肢であり、どちらかを採取すると共通副目標を達成する
- 辺境斥候の灰地踏破能力は浅瀬へ適用されず、移動コスト2のまま
- 将来の飛行兵は浅瀬を移動コスト1として扱う
- 浅瀬と薬草地点が同じマスになる場合、薬草地点の取得判定と浅瀬の移動コスト2を両方適用する
- 茂みの回避は敵味方の双方へ適用し、攻撃側の所属や武器種では変化しない
- 茂み上への攻撃結果は遠征seedと攻撃順から決定し、同じ戦闘状態では再現可能にする

### 1. 灰枝の林縁

判断テーマ: 情報か速度か。探索選択、通常戦闘、副目標を最初に検証する地点。

敵編成:

- 狼4体
- 急行ルートのみ狼3体

探索選択:

| 選択 | 敵 | 情報・配置 | 事前消耗 | 報酬補正 |
|---|---:|---|---|---|
| 痕跡を調べる | 4体 | 一部配置公開、通常配置 | なし | 通常 |
| 急いで奥へ進む | 3体 | 非公開、通常配置 | 生存中の味方全員HP-2、最低HP1 | 通常勝利の木材なし |
| `[辺境斥候]` 獣道を調査 | 4体 | 全公開、左3列で自由配置 | なし | 獣皮+1 |

主目的:

- 狼を全滅させる

副目標「踏査地点を確保」:

- 盤面右側に踏査地点を1マス配置
- 味方が踏査地点上で行動終了すると調査済みになる
- 到達しただけでは達成しない
- 調査済み状態で戦闘に勝利すると追加報酬とObjectiveを得る

報酬:

| 条件 | Pending Loot / Progress |
|---|---|
| 通常勝利 | 木材2、獣皮1 |
| 急行ルート勝利 | 木材0、獣皮1 |
| 踏査地点確保＋勝利 | 木材+1、`ashbough_verge_surveyed` |
| 斥候ルート＋勝利 | 獣皮+1 |

#### 狼AI

狼は個別に最短距離で突撃せず、群れとして同じ対象を選ぶ。

- 単独では味方の攻撃範囲へ無理に進入しない
- 2体以上が同じ対象へ接近可能な場合、集団攻撃を開始
- 対象優先順位:
  1. 周囲の味方が少ない孤立ユニット
  2. 現在HPが低いユニット
  3. 距離が近いユニット
  4. UnitId順
- 同じ状態では必ず同じ判断になるよう決定論的に処理する

斥候の高移動力には、戦闘だけでなく踏査地点へ先行する用途を持たせる。

#### 狼の初期戦闘値

| 項目 | 値 |
|---|---:|
| HP | 16 |
| STR | 6 |
| MAG | 0 |
| SPD | 7 |
| DEF | 2 |
| RES | 1 |
| MOV | 5 |
| 武器 | 牙（物理、威力5、射程1） |

- 狼は全地点で同じ基本値を使用し、地点ごとの能力補正を付けない
- 狼は茂み、浅瀬、灰地の通常効果を受ける。特殊な地形無視能力は持たない
- 狼の通常攻撃に毒、出血、ノックバック、反撃不能などの追加効果を付けない
- 最低ダメージ1の共通規則を適用する

### 2. 薬草の沢

判断テーマ: 回復資源を得るために、どこまで戦闘上の危険を受け入れるか。

基本盤面:

- 初期敵は狼5体
- 盤面中央に浅瀬と薬草地点2マス
- 浅瀬の移動コストは2
- 辺境斥候は浅瀬ペナルティを無視しない
- 将来、飛行兵のみ浅瀬を無視できる

探索選択:

| 選択 | 資源 | 戦闘への影響 | 地域進行 |
|---|---|---|---|
| そのまま通過 | なし | 狼5体、通常配置、増援なし | なし |
| 薬草を採取 | 薬草2 | 狼5体、2ターン目に狼1体増援 | なし |
| `[暁の衛生兵]` 薬草を選別 | 高品質薬草1 | 増援なし、味方全員を左2列のランダム候補へ制限 | 専用調査を有効化 |

主目的:

- 狼を全滅させる

共通副目標「薬草地点確保」:

- 任意の味方が薬草地点上で行動終了
- その後に戦闘勝利
- 薬草1を追加

衛生兵専用踏査:

- 衛生兵ルートを選択すると専用の調査地点を表示
- 暁の衛生兵本人がその地点上で行動終了する
- 戦闘勝利時に`herbwater_hollow_surveyed`をRegionProgressへ記録
- 選択しただけ、または他兵種が地点で待機しただけでは達成しない

報酬:

| 条件 | Pending Loot / Progress |
|---|---|
| 通常勝利 | 木材1 |
| 薬草採取ルート＋勝利 | 薬草2 |
| 衛生兵ルート＋勝利 | 高品質薬草1 |
| 共通薬草地点確保＋勝利 | 薬草+1 |
| 衛生兵専用踏査＋勝利 | `herbwater_hollow_surveyed` |

増援ルール:

- 薬草採取ルートでのみ発生
- 2ターン目のEnemy Phase開始時に狼1体
- 右端の通行可能な予定マスを戦闘開始時から表示
- 予定マスが塞がれていれば同じ列の別マスへ出現
- 同じ列の候補がすべて塞がれていれば増援なし

この地点の後に最初のキャンプを置き、得た資源と残りHPを見て続行か帰還かを選ばせる。

このキャンプは`base_development.md`の既存ルールに従い、自動全回復は行わない。回復手段は
救急セット、野戦治療キット、野営食、救命包などの所持アイテム消費のみで、拠点への安全帰還時のみ
全回復する。灰枝の森だからといって例外の全回復枠は設けない。

初回の薬草地点確保後、帰還せず「遠征を続ける」を選んだ場合だけ、採取直後の新鮮な薬草を使って
生存者全員のHPを2回復する。1遠征1回で、薬草Lootを消費せず、HP満タンのUnitには効果を与えない。
これは第1周でBossへ続行する小さな利点であり、安全帰還後や安全通過時には発生しない。

### 3. 折れ木の縄張り

判断テーマ: 純粋な戦闘、地形利用、情報優位のどれでボスへ挑むか。

探索選択:

| ルート | 戦闘効果 | 最終報酬 |
|---|---|---|
| 慎重に縄張りへ入る | 通常倒木1本、標準予告 | 木材3、獣皮2、大牙1 |
| 倒木を戦場へ誘導する | 誘導可能な候補へ倒木2本をランダム配置 | 木材2、獣皮2、大牙1 |
| `[辺境猟兵]` 獣の痕跡を追う | ボスの次回行動を常時先読み | 木材3、獣皮3、大牙1 |

報酬内訳:

- ボス撃破保証: 灰角の大牙1、木材2、獣皮2
- 通常倒木を回収: 木材+1（A、C）
- Bでは倒木を罠として破損させるため追加木材なし
- Cでは追跡中に発見した古い獣道から獣皮+1
- 辺境猟兵は灰枝の森攻略後の加入候補であるため、Cは初回攻略用ではなく再訪・再挑戦用の選択肢とする

### ランダム初期配置

- 味方4人は左2列の通行可能な候補から重複なしで生成する
- 灰角大猪は右2列の候補から生成する
- 護衛狼4体を大猪と重ならない敵側候補へ生成する
- 通常倒木1本は中央4列（col 2-5）の通行可能な候補へ配置する
- Bルートでは異なる行を優先して倒木をもう1本配置する
- ボスと倒木を開始時に同じ突進行へ置かず、初手から確定スタンを取れないようにする
- ボスから味方までの突進可能行が最低1つ生まれる配置を保証する
- 生成後に、味方が移動によって倒木へ誘導できる経路が存在することを検証する

固定されるのは左右の陣営範囲と上記の成立条件だけで、行、正確な列、通常地形は遠征ごとに変わる。

### 予告情報

- A、B: ボスが突進を決定した後、実行1ターン前に攻撃列を表示
- C: 決定前から次に狙う味方、次回行動、攻撃列、停止予定位置を表示
- C: 激昂するHP境界も表示
- 予告後に対象が移動しても列を変更せず、追尾しない

Boss戦開始時、次を一度だけ戦闘メッセージとして表示する。

1. 「突進する列は予告後に固定されます」
2. 「倒木へ衝突させると、灰角大猪は次の敵フェイズを失います」
3. 「HPが半分以下になると激昂し、突進距離が伸びます」

同じ説明を各Roundで繰り返さず、以後は実際の突進列、衝突、激昂発生だけを通知する。

## ボス仕様

### 灰角大猪

個体呼称: 折れ木の主

稀に大きく育った猪。悪意ある魔物ではなく、人間が街道予定地へ入ったことで縄張り争いが
起きている。灰をかぶって白く変色した大牙を、開拓民が「灰角」と呼んでいる。

基本値:

| 項目 | 通常 | 激昂後 |
|---|---:|---:|
| HP | 140 | 140 |
| STR | 9 | 11 |
| DEF | 5 | 5 |
| RES | 1 | 1 |
| MOV | 2 | 2 |
| 突進距離 | 3 | 4 |
| 突進威力補正 | +50 | +50 |
| 薙ぎ払い威力補正 | +2 | +2 |

- 1マス占有
- ノックバック無効
- 通常命中100%。防御側が茂みにいる場合のみ命中80%
- HP50%以下で激昂を一度だけ発動
- 護衛狼4体を戦闘開始時から配置する。大猪が倒木で行動を失っても、護衛狼は通常の群れAIで行動する

### 行動優先順位

1. HP50%以下で未発動なら激昂
2. 予告済みの突進があれば実行
3. 前方3マスに味方が1体以上いれば薙ぎ払い。複数いる場合は全員へ命中
4. 突進可能な対象がいれば突進を予告
5. 激昂後で突進可能な対象がいなければ、味方と同じ行へMOV2で位置調整して突進を予告
6. 激昂前で上記を満たさなければMOV2で位置調整

激昂は状態更新であり行動を消費しない。激昂したEnemy Phaseから、予告済み突進の実行、
薙ぎ払い、位置調整を伴う突進予告のいずれかを続けて行う。

### 突進

- 同じ行を通常3マス、激昂後4マス進む
- 実行1ターン前に攻撃列を予告
- 予告後は列を固定
- 通過した味方全員へ`STR + 50 - DEF`ダメージ。予告を無視した場合は原則として戦闘不能
- 味方を通過しても終点まで進む
- 倒木または盤端で停止

### 薙ぎ払い

向いている側の前上、正面、前下を同時攻撃する。盤外マスは無視する。

```text
[対象][対象][対象]
       [猪]
```

- 各対象へ`STR + 2 - DEF`ダメージ
- 前衛を同じ側へ固めると複数人が被弾する

### 倒木衝突

- 突進を即時中断
- 衝突した倒木を破壊
- 次のEnemy Phaseの行動を完全に失う
- スタン中はDEF 2、RES 0
- 次に行動可能になる直前にDEF 5、RES 1へ戻る

ボス戦で教える判断:

- 予告列から逃げる
- 倒木へ誘導する
- 弱体化中に集中攻撃する
- 低RESを魔法で狙う

ボス戦は3x8盤面の直線、障害物、位置取りを最初に学ぶ戦闘とする。

初期実装での標準結果は一律「負傷した大猪が森の奥へ退き、街道予定地から縄張りが外れる」で
確定する。討伐か誘導かをルートで演出上分岐させる仕様は今回のスコープに含めず、将来の追加
コンテンツへ先送りする。探索ルートによる結果の違いは、既存の報酬表（木材・獣皮・大牙の
数量差）だけで表現する。

## 初期バランス値と検証基準

現行実装に対するシード別の計測結果は
[`../balance_test_results.md`](../balance_test_results.md)へ記録する。設計値と実装データが一致するまでは、
計測結果を根拠にこの表の数値を自動的に変更しない。

実装開始時は次の値を採用し、実装者判断で先に変更しない。

| 対象 | 初期値 |
|---|---:|
| 林縁の狼 | 通常4体、急行3体 |
| 狼基本値 | HP16、STR6、DEF2、RES1、MOV5、牙威力5・射程1 |
| 急行の事前ダメージ | 生存者全員2、最低HP1 |
| 薬草の沢の狼 | 初期5体、採取時増援1体 |
| 薬草地点の回復 | HP5 |
| 灰角大猪 | HP140、STR9、DEF5、RES1、MOV2 |
| 護衛狼 | 戦闘開始時4体。大猪の倒木スタン中も通常行動 |
| 激昂後 | STR11、突進距離4 |
| 突進補正 | +50。予告列を無視した場合は原則として戦闘不能になる |
| 薙ぎ払い補正 | +2 |
| 倒木衝突 | 次の敵Phase喪失、DEF2、RES0 |

検証は初期4人、初期武器、拠点強化なしを基準に行う。

| 戦闘 | 合格基準 |
|---|---|
| 灰枝の林縁 | 標準ルートで初見全滅を常態化させず、孤立すると明確に危険になる |
| 薬草の沢 | 採取ルートの増援が薬草2個分の価値に見合う消耗を発生させる |
| 灰角大猪 | 予告を見て回避すれば無被害にでき、無視すれば複数被弾する |
| ボス戦時間 | 標準編成で4〜7ラウンドを目安とする |
| 倒木利用 | 衝突成功で少なくとも味方1人分以上の攻撃機会を得る |
| 魔法役 | 低RESを狙う価値があるが、魔法役なしでも攻略可能 |

最低30Seedで生成不変条件を自動検証し、手動では各探索ルートを最低3回ずつ確認する。
調整が必要な場合は、最初に敵配置とAI評価、次に敵数、最後に能力値の順で変更する。
プレイヤー能力値を灰枝の森専用に補正しない。

灰枝の林縁は、狼の数値を下げて安全にするのではなく、隊形と兵種運用によって被害が変わる導入戦にする。

| 操作水準 | 戦闘不能が発生する戦闘の目安 |
|---|---:|
| 単純な最短接近・集中攻撃 | 25～35% |
| 古参守備兵の進路封鎖、孤立回避、弓の先制、衛生兵保護を使う | 10～20% |
| 孤立、衛生兵の前出し、攻撃範囲の無視など重大な判断ミス | 40%以上 |

初回・消耗品なし・3地点連戦の全滅率は、操作水準を混ぜて30%（許容25～35%）を目標とする。予告回避、
倒木誘導、回復資源の使用によって明確に下げられることを条件とし、戦闘不能率を一律に下げる調整は行わず、
戦術的な操作を使った場合に数値が下がることを確認する。差が出ない場合は狼の能力値ではなく、AIの
対象評価、初期配置、地形生成、味方兵種能力が実際に機能しているかを先に見直す。

## 重要発見と通常戦利品

### 地域進行

踏査進行は加算値やDiscoveryとして直接管理せず、達成済みObjective IDを保持する。

```cpp
struct RegionProgress {
    RegionId region;
    std::unordered_set<ObjectiveId> completedObjectives;
    bool bossDefeated = false;
};
```

灰枝の森で使用する条件:

- `ashbough_verge_surveyed`
- `herbwater_hollow_surveyed`
- `bossDefeated == true`

3条件がすべて揃った戦闘勝利時に限り、
`ashbough_forest_survey_complete`をPending Discoveriesへ追加する。安全帰還後にDiscovery
Registryへ恒久登録する。同じObjectiveやDiscoveryは重複登録しない。

Pending Discoveries:

- 森の踏査記録: `ashbough_forest_survey_complete`成立後の戦闘勝利でPendingへ追加し、次地域への安全なルートを確定
- 薬草群生地: `herbwater_hollow_surveyed`成立後の戦闘勝利でPendingへ追加し、診療所「野戦医療」系の解放条件にする

灰角の大牙はDiscoveryではなく、拠点発展で消費するキー素材としてPending Lootへ入れる。

Pending Loot候補:

- 木材
- 薬草
- 高品質薬草
- 獣皮
- 灰角の大牙

重要発見は該当条件を満たして地点を攻略した後にPending Discoveriesへ入り、安全帰還時に
Discovery Registryへ登録される。

### 報酬付与台帳

- 報酬は`expeditionAttemptId + ObjectiveId + RewardId`を一意キーとして1回だけPendingへ追加する
- 探索ルートの基本報酬は`expeditionAttemptId + SiteId + routeId`を一意キーとする
- 戦闘勝利前には通常素材、重要発見、地域進行のどれも追加しない
- 戦闘勝利時に基本報酬、Completed副目標報酬、地域進行を同じMissionFlow処理でPendingへ追加する
- 敗北と手動撤退では、その遠征で追加された灰枝の森のPending報酬をすべて破棄する
- 安全帰還でStorage、Discovery Registry、恒久RegionProgressへ確定する
- 安全通過では報酬を追加しない。危険区域の再調査では再訪許可された通常素材だけを付与する
- 灰角の大牙、灰角の欠片、Discovery、初回Objective報酬は恒久取得後に再取得できない

## 攻略後の変化

```text
森の通行路を確保
-> 木材と薬草の供給開始
-> 恒久的な木造施設を建設可能
-> 野営地から開拓拠点へ発展可能
-> 森の向こうの沈黙した監視所群を遠征先として解放
```

拠点外観も段階的に変化する。

- テントと焚き火
- 木材の搬入
- 最初の木造施設
- 新しい開拓民の到着

### 開拓拠点への発展

必要条件:

- 灰角の大牙1
- 木材3
- 灰角大猪を撃破して安全帰還

ボス撃破で木材2を保証し、林縁または薬草の沢の最低報酬と合わせて、どの探索ルートでも
地域クリア後に発展不能にならないようにする。

第1段階では常時利用可能な基礎設備と、素材を使って建設する施設を分ける。

基礎設備（枠を消費しない）:

- 作戦テント: 遠征先選択
- 倉庫: Storage管理
- 共同テント: 仲間と4人編成

建設施設:

- 救護テント、訓練場、工作台、簡易鍛冶台を素材が揃った順に建設できる
- 建設後は恒久的に残り、切替なしで常時利用できる
- Discovery、解放技術、製作済み装備は失わない
- 同時稼働枠と停止状態は設けない

| 施設 | 建設費 | 条件 | 第1段階の効果 |
|---|---|---|---|
| 救護テント | 木材2、薬草2 | 薬草群生地を発見 | 応急手当具と救命包の製作 |
| 訓練場 | 木材3、獣皮2 | なし | 基礎兵種技の装備変更 |
| 工作台 | 木材3、獣皮1 | なし | 防護板と基本探索道具の製作 |
| 簡易鍛冶台 | 木材2、獣皮1 | なし | 武器変更、調整特性1枠、分岐レシピ閲覧 |

簡易鍛冶台の初期調整は「獣皮の柄巻き」のみ。獣皮1を消費し、戦闘ごとに最初に受ける
ノックバックを1回無効化する。武器耐久と通常修理は採用しない。

### 地域全体のゲーム進行

```text
初期野営地で4人編成・6枠準備
-> 灰枝の林縁（情報選択、狼戦、踏査副目標）
-> 薬草の沢（資源選択、増援、衛生兵踏査）
-> キャンプ（続行または安全帰還）
-> 折れ木の縄張り（ボス攻略方法を選択）
-> 灰角大猪戦
-> Pending Loot、RegionProgress、Pending Discoveriesを獲得
-> 安全帰還
-> LootをStorage、DiscoveriesをRegistryへ確定

```

## セーブとの関係

灰枝の森専用のセーブ設計は不要とする。`docs/save_system.md`の「後期実装の確定仕様 >
遠征中断セーブ」がすでに`RegionProgress`、Pending Loot、Pending Discoveries、生成済み地形
スナップショット、地域ID／地点IDを汎用チェックポイント項目として定義しており、本書の
`RegionProgress`とPendingリストはその仕様にそのまま乗る。

現行の簡略版遠征中断セーブは探索開始時とキャンプ時を保存し、戦闘途中と配置途中は保存しない。
戦闘中に終了した場合は直前の探索開始時またはキャンプへ戻り、同じ遠征Seedから盤面を再生成する。
今後`SiteAccessState`をSchemaへ追加し、安全帰還で確定した地点状態を恒久保存する。

地域ID／地点IDは、seed生成（本書「戦闘マップ生成共通ルール」）とセーブ側チェックポイントの
両方で同じ安定文字列を使う。この地域では次のIDを正式採用する。

| 対象 | 安定ID |
|---|---|
| 地域 | `ashbough_forest` |
| 灰枝の林縁 | `ashbough_verge` |
| 薬草の沢 | `herbwater_hollow` |
| 折れ木の縄張り | `broken_bough_territory` |

`ashbough_verge_surveyed`、`herbwater_hollow_surveyed`、`ashbough_forest_survey_complete`などの
ObjectiveId／DiscoveryIdは、上記の地点IDをそのまま接頭辞として使っており、変更しない。
