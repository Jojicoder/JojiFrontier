# 第1地域案: 灰枝の森 / Ashbough Forest

## 状態

JOJIFrontierの第1地域として検討中のシナリオ案。名称は未確定。

参照したWorld Bible:

- `../../../JojiWorldBible/docs/world/embermarch.md`
- `../../../JojiWorldBible/docs/world/geography.md`
- `../../../JojiWorldBible/locations.md`
- `../../../JojiWorldBible/naming.md`
- `../../../JojiWorldBible/docs/world/flora_fauna.md`

### 正史上の所属

- 大陸: Asteria
- 国: Embermarch王国
- マクロ地域: South / Southern Ashlands
- 前線: Embermarch南端のMarch Line周辺

EmbermarchはHouse Bloodmarchが統治するAggressive / Military国家で、南方の未踏領域に
対してMarch Lineを維持している。JOJIFrontierのプレイヤーはWarmarchそのものではなく、
前線拠点を運営するExpedition Marshalである。

### 正史との整合

World Bibleの`flora_fauna.md`では、普通の捕食動物、縄張りを持つ動物、追いやられた群れ、
稀な大型個体が街道や辺境開拓地で戦闘上の脅威になり得る。これは魔法生物の常設ベスティアリや
国家規模の怪物狩り組織を意味しない。

灰角大猪はこのルールに沿う稀な大型個体とする。魔物や未知の固有種ではなく、火災、乾燥、
軍の往来、新規開拓による生息域の変化で人間と衝突した普通の猪である。March Line外に何が
存在するかという意図的な空白は変更しない。

## 名称重複確認

2026-07-12時点で、World BibleとJOJIFrontier内に次の既存名称との完全一致はない。

- 灰枝の森 / Ashbough Forest
- 灰枝の林縁 / Ashbough Verge
- 薬草の沢 / Herbwater Hollow
- 折れ木の縄張り / Broken-Bough Territory
- 灰角大猪 / Ash-Tusk Boar
- 折れ木の主 / Lord of the Broken Boughs

Embermarchの既存命名はBloodmarch、Ironwatch、Scorchreachなど、軍事・火・前線を表す
複合語が中心である。Ashboughは灰と森林を組み合わせた地名として大きく逸脱しないが、
正式名称への確定はWorld Bible側の命名承認後とする。

## 地域の目的

拠点の先に広がる森林で大型獣の活動により街道予定地が使えない。

- 調査隊が襲われている
- 木材や薬草を採取できない
- 安全な水場を確保できない
- 開拓民を送り込めない

目的は森の征服ではなく、安全な通行路の確保である。

## 地域構造

最初の短い導入遠征として3地点で構成する。

1. 灰枝の林縁 / Ashbough Verge
2. 薬草の沢 / Herbwater Hollow
3. 折れ木の縄張り / Broken-Bough Territory

地点ごとの役割を情報、資源、ボス条件に分け、同じ判断を繰り返さない。

### 1. 灰枝の林縁

判断テーマ: 情報か速度か。探索選択、通常戦闘、副目標を最初に検証する地点。

敵編成:

- 狼4体
- 急行ルートのみ狼3体

探索選択:

| 選択 | 敵 | 情報・配置 | 事前消耗 | 報酬補正 |
|---|---:|---|---|---|
| 痕跡を調べる | 4体 | 一部配置公開、通常配置 | なし | 通常 |
| 急いで奥へ進む | 3体 | 非公開、通常配置 | 生存中の味方全員HP-2、最低HP1 | 通常勝利の木材なし |
| `[Frontier Scout]` 獣道を調査 | 4体 | 全公開、左3列で自由配置 | なし | 獣皮+1 |

主目的:

- 狼を全滅させる

副目標「踏査地点を確保」:

- 盤面右側に踏査地点を1マス配置
- 味方が踏査地点上で行動終了すると調査済みになる
- 到達しただけでは達成しない
- 調査済み状態で戦闘に勝利すると追加報酬とObjectiveを得る

報酬:

| 条件 | Pending Loot / Progress |
|---|---|
| 通常勝利 | 木材2、獣皮1 |
| 急行ルート勝利 | 木材0、獣皮1 |
| 踏査地点確保＋勝利 | 木材+1、`ashbough_verge_surveyed` |
| 斥候ルート＋勝利 | 獣皮+1 |

#### 狼AI

狼は個別に最短距離で突撃せず、群れとして同じ対象を選ぶ。

- 単独では味方の攻撃範囲へ無理に進入しない
- 2体以上が同じ対象へ接近可能な場合、集団攻撃を開始
- 対象優先順位:
  1. 周囲の味方が少ない孤立ユニット
  2. 現在HPが低いユニット
  3. 距離が近いユニット
  4. UnitId順
- 同じ状態では必ず同じ判断になるよう決定論的に処理する

斥候の高移動力には、戦闘だけでなく踏査地点へ先行する用途を持たせる。

### 2. 薬草の沢

判断テーマ: 回復資源を得るために、どこまで戦闘上の危険を受け入れるか。

基本盤面:

- 初期敵は狼3体
- 盤面中央に浅瀬と薬草地点2マス
- 浅瀬の移動コストは2
- Frontier Scoutは浅瀬ペナルティを無視しない
- 将来、飛行兵のみ浅瀬を無視できる

探索選択:

| 選択 | 資源 | 戦闘への影響 | 地域進行 |
|---|---|---|---|
| そのまま通過 | なし | 狼3体、通常配置、増援なし | なし |
| 薬草を採取 | 薬草2 | 狼3体、2ターン目に狼1体増援 | なし |
| `[Dawn Chirurgeon]` 薬草を選別 | 高品質薬草1 | 増援なし、味方全員を左2列の通常配置へ固定 | 専用調査を有効化 |

主目的:

- 狼を全滅させる

共通副目標「薬草地点確保」:

- 任意の味方が薬草地点上で行動終了
- その後に戦闘勝利
- 薬草1を追加

衛生兵専用踏査:

- 衛生兵ルートを選択すると専用の調査地点を表示
- Dawn Chirurgeon本人がその地点上で行動終了する
- 戦闘勝利時に`herbwater_hollow_surveyed`をRegionProgressへ記録
- 選択しただけ、または他兵種が地点で待機しただけでは達成しない

報酬:

| 条件 | Pending Loot / Progress |
|---|---|
| 通常勝利 | 木材1 |
| 薬草採取ルート＋勝利 | 薬草2 |
| 衛生兵ルート＋勝利 | 高品質薬草1 |
| 共通薬草地点確保＋勝利 | 薬草+1 |
| 衛生兵専用踏査＋勝利 | `herbwater_hollow_surveyed` |

増援ルール:

- 薬草採取ルートでのみ発生
- 2ターン目のEnemy Phase開始時に狼1体
- 右端の通行可能な予定マスを戦闘開始時から表示
- 予定マスが塞がれていれば同じ列の別マスへ出現
- 同じ列の候補がすべて塞がれていれば増援なし

この地点の後に最初のキャンプを置き、得た資源と残りHPを見て続行か帰還かを選ばせる。

### 3. 折れ木の縄張り

判断テーマ: 純粋な戦闘、地形利用、情報優位のどれでボスへ挑むか。

探索選択:

| ルート | 戦闘効果 | 最終報酬 |
|---|---|---|
| 慎重に縄張りへ入る | 通常倒木1本、標準予告 | 木材3、獣皮2、大牙1 |
| 倒木を戦場へ誘導する | 有利な固定位置に倒木2本 | 木材2、獣皮2、大牙1 |
| `[Frontier Ranger]` 獣の痕跡を追う | ボスの次回行動を常時先読み | 木材3、獣皮3、大牙1 |

報酬内訳:

- ボス撃破保証: 灰角の大牙1、木材2、獣皮2
- 通常倒木を回収: 木材+1（A、C）
- Bでは倒木を罠として破損させるため追加木材なし
- Cでは追跡結果として獣皮+1

### 固定初期配置

現在の座標系に合わせて味方を左側、ボスを右側に配置する。

```text
列   0  1  2  3  4  5  6  7

上  [P][ ][ ][ ][T][ ][ ][ ]
中  [P][P][ ][ ][ ][ ][B][ ]
下  [ ][P][ ][ ][ ][ ][ ][ ]
```

- `P`: 味方
- `B`: 灰角大猪 (`row 1, col 6`)
- `T`: 通常倒木 (`row 0, col 4`)
- 味方4人は自陣2列以内の固定配置

Bルートでは追加倒木を`row 2, col 5`へ置く。ボスと倒木を開始時に別行へ置き、初手から
確定で衝突させられないようにする。味方の位置を調整して突進行を上下へ誘導する必要がある。

### 予告情報

- A、B: ボスが突進を決定した後、実行1ターン前に攻撃列を表示
- C: 決定前から次に狙う味方、次回行動、攻撃列、停止予定位置を表示
- C: 激昂するHP境界も表示
- 予告後に対象が移動しても列を変更せず、追尾しない


## ボス仕様

### 灰角大猪 / Ash-Tusk Boar

個体呼称: 折れ木の主 / Lord of the Broken Boughs

稀に大きく育った猪。悪意ある魔物ではなく、人間が街道予定地へ入ったことで縄張り争いが
起きている。灰をかぶって白く変色した大牙を、開拓民が「灰角」と呼んでいる。

基本値:

| 項目 | 通常 | 激昂後 |
|---|---:|---:|
| HP | 48 | 48 |
| STR | 8 | 10 |
| DEF | 5 | 5 |
| RES | 1 | 1 |
| MOV | 2 | 2 |
| 突進距離 | 3 | 4 |
| 突進威力補正 | +4 | +4 |
| 薙ぎ払い威力補正 | +2 | +2 |

- 1マス占有
- ノックバック無効
- 基本必中
- HP50%以下で激昂を一度だけ発動

### 行動優先順位

1. HP50%以下で未発動なら激昂
2. 予告済みの突進があれば実行
3. 前方3マスに複数の味方がいれば薙ぎ払い
4. 突進可能な対象がいれば突進を予告
5. それ以外はMOV2で位置調整

### 突進

- 同じ行を通常3マス、激昂後4マス進む
- 実行1ターン前に攻撃列を予告
- 予告後は列を固定
- 通過した味方全員へ`STR + 4 - DEF`ダメージ
- 味方を通過しても終点まで進む
- 倒木または盤端で停止

### 薙ぎ払い

向いている側の前上、正面、前下を同時攻撃する。盤外マスは無視する。

```text
[対象][対象][対象]
       [猪]
```

- 各対象へ`STR + 2 - DEF`ダメージ
- 前衛を同じ側へ固めると複数人が被弾する

### 倒木衝突

- 突進を即時中断
- 衝突した倒木を破壊
- 次のEnemy Phaseの行動を完全に失う
- スタン中はDEF 2、RES 0
- 次に行動可能になる直前にDEF 5、RES 1へ戻る

ボス戦で教える判断:

- 予告列から逃げる
- 倒木へ誘導する
- 弱体化中に集中攻撃する
- 低RESを魔法で狙う

ボス戦は3x8盤面の直線、障害物、位置取りを最初に学ぶ戦闘とする。

撃破は必ずしも殺害を意味しない。標準結果では負傷した大猪が森の奥へ退き、街道予定地から
縄張りが外れる。将来、猟兵ルートでは殺さず誘導し、正面ルートでは討伐となる結果差を
検討できる。

## 重要発見と通常戦利品

### 地域進行

踏査進行は加算値やDiscoveryとして直接管理せず、達成済みObjective IDを保持する。

```cpp
struct RegionProgress {
    RegionId region;
    std::unordered_set<ObjectiveId> completedObjectives;
    bool bossDefeated = false;
};
```

灰枝の森で使用する条件:

- `ashbough_verge_surveyed`
- `herbwater_hollow_surveyed`
- `bossDefeated == true`

3条件がすべて揃った戦闘勝利時に限り、
`ashbough_forest_survey_complete`をPending Discoveriesへ追加する。安全帰還後にDiscovery
Registryへ恒久登録する。同じObjectiveやDiscoveryは重複登録しない。

Pending Discoveries:

- 森の踏査記録: 次地域への安全なルートを確定
- 薬草群生地: 診療所「野戦医療」系の解放条件

灰角の大牙はDiscoveryではなく、拠点発展で消費するキー素材としてPending Lootへ入れる。

Pending Loot候補:

- 木材
- 薬草
- 高品質薬草
- 獣皮
- 灰角の大牙

重要発見は該当条件を満たして地点を攻略した後にPending Discoveriesへ入り、安全帰還時に
Discovery Registryへ登録される。

## 攻略後の変化

```text
森の通行路を確保
-> 木材と薬草の供給開始
-> 恒久的な木造施設を建設可能
-> 野営地から開拓拠点へ発展可能
-> 森の向こうのCinderwatch Gateを遠征先として解放
```

拠点外観も段階的に変化する。

- テントと焚き火
- 木材の搬入
- 最初の木造施設
- 新しい開拓民の到着

### 開拓拠点への発展

必要条件:

- 灰角の大牙1
- 木材3
- 灰角大猪を撃破して安全帰還

ボス撃破で木材2を保証し、林縁または薬草の沢の最低報酬と合わせて、どの探索ルートでも
地域クリア後に発展不能にならないようにする。

第1段階では常時利用可能な基礎設備と、有効枠を消費する施設を分ける。

基礎設備（枠を消費しない）:

- 作戦テント: 遠征先選択
- 倉庫: Storage管理
- 共同テント: 仲間と4人編成

有効施設:

- 有効枠は2
- 救護テント、訓練場、工作台、簡易鍛冶台から選択
- 解体可能
- 解体時は各使用素材の50%を端数切り捨てで返却
- Discovery、解放技術、製作済み装備は失わない
- 次の開拓段階で有効枠を増やす

| 施設 | 建設費 | 条件 | 第1段階の効果 |
|---|---|---|---|
| 救護テント | 木材2、薬草2 | 薬草群生地を発見 | 応急手当具と救命包の製作 |
| 訓練場 | 木材3、獣皮2 | なし | 基礎兵種技の装備変更 |
| 工作台 | 木材3、獣皮1 | なし | 防護板と基本探索道具の製作 |
| 簡易鍛冶台 | 木材2、獣皮1 | なし | 武器変更、調整特性1枠、分岐レシピ閲覧 |

簡易鍛冶台の初期調整は「獣皮の柄巻き」のみ。獣皮1を消費し、戦闘ごとに最初に受ける
ノックバックを1回無効化する。武器耐久と通常修理は採用しない。

### 地域全体のゲーム進行

```text
初期野営地で4人編成・6枠準備
-> 灰枝の林縁（情報選択、狼戦、踏査副目標）
-> 薬草の沢（資源選択、増援、衛生兵踏査）
-> キャンプ（続行または安全帰還）
-> 折れ木の縄張り（ボス攻略方法を選択）
-> 灰角大猪戦
-> Pending Loot、RegionProgress、Pending Discoveriesを獲得
-> 安全帰還
-> LootをStorage、DiscoveriesをRegistryへ確定
-> 大牙1と木材3で開拓拠点へ発展
-> 4候補から有効施設2つを建設
-> Cinderwatch Gate方面を解放
```

## 既存シナリオとの関係

現行の`docs/frontier_setting.md`では、Cinderwatchはすでに奪還済みの初期拠点であり、
Silent Postsが最初の遠征になっている。灰枝の森を第1地域として採用すると、次のように
順序を改訂する必要がある。

```text
初期野営地
-> 第1地域: 灰枝の森
-> Cinderwatchへの道を確保
-> Cinderwatchを開拓拠点として利用
-> 第2地域: Silent Posts / Cinderwatch Gate方面
```

これは既存のFrontierローカル設定の変更であり、World BibleのRetconではない。正式採用時に
`frontier_setting.md`、`base_development.md`、`exploration_system.md`、README、ゲームデータの
順序をまとめて更新する。
