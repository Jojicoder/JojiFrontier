# 商店仕様

文書種別: **正本**
正本範囲: 商店で購入できるもの、購入禁止品、在庫、価格方針、施設との役割分担

状態: JOJIFrontier固有の拠点機能として採用。消耗品と探索道具の効果は
[`item_system.md`](item_system.md)、倉庫上限と受取保留は
[`inventory_overflow.md`](inventory_overflow.md)、施設研究と製作は
[`base_development.md`](base_development.md)と[`facility_research.md`](facility_research.md)、
所持品分類の参照一覧は[`item_catalog.md`](item_catalog.md)を参照する。

共有世界の経済方針は`../JojiWorldBible/docs/world/economy.md`を参照する。商店は遠征テンポを
補助する拠点機能であり、地域探索、施設研究、鍛冶、工房、診療所の価値を置き換えない。

## 役割

商店の役割は、遠征前の最低限の補給と、少し足りない基礎素材の補助である。

- 主な成長は遠征で素材とDiscoveryを持ち帰ることで進む
- 商店は進行必須のDiscovery、キー素材、分岐武器、上位素材を売らない
- 商店だけで拠点発展、兵種解放、武器分岐、地域解放を完結できない
- プレイヤーが遠征失敗後に完全に詰まらないよう、基本消耗品だけは安定供給する

## 通貨

通貨名は共有世界の標準銀貨に合わせ、表示名を「クラウン」とする。

```cpp
using Money = int;
```

- クラウンは安全帰還時の報酬、依頼、地域攻略、余剰通常素材の将来売却で得る
- 初期実装では売却を採用しない場合、クラウン報酬は戦闘結果や地域結果から直接付与する
- Discovery、キー素材、装備中武器、唯一の探索道具は売却不可
- 素材売却を導入する場合も通常素材だけに限定し、価格は購入価格より低くする

## 販売区分

| 区分 | 販売可否 | 方針 |
|---|---|---|
| 基本消耗品 | 可 | 常時在庫。遠征失敗後の立て直し用 |
| 解放済み消耗品 | 可 | 対応施設研究後に入荷 |
| 基礎素材 | 条件付き可 | 少量・高価格・在庫制限あり |
| 情報商品 | 可 | 司令所研究と連動。情報だけを売る |
| 便利品 | 可 | 遠征準備を補助するが、戦闘能力を直接上げない |
| 探索道具 | 原則不可 | 工房製作で取得。商店は完成品を売らない |
| 武器本体 | 不可 | 鍛冶場の役割を守る |
| 分岐武器 | 不可 | Discoveryと鍛冶研究の価値を守る |
| 調整特性 | 不可 | 鍛冶場製作で取得 |
| 高品質素材 | 不可 | 副目標、危険ルート、後半地域の価値を守る |
| キー素材 | 不可 | 地域攻略の証明として扱う |
| 重要発見 | 不可 | 知識・記録であり売買しない |
| 仲間・兵種解放 | 不可 | 戦略の核を商店で買わせない |

## 販売品

### 常時販売

| ID | 日本語名 | 入荷条件 | 在庫 | 価格方針 |
|---|---|---|---:|---|
| `first_aid_kit` | 救急セット | 商店解放 | 3 | 高め。キャンプ専用の大回復 |
| `field_treatment_kit` | 野戦治療キット | 商店解放 | 4 | 標準。戦闘中にも使える小回復 |
| `rescue_pack` | 救命包 | 商店解放 | 1 | 高め。戦闘不能対策 |
| `camp_rations` | 野営食 | 開拓拠点 | 2 | 標準。長期遠征の補助 |

常時販売品は遠征から帰還するたびに在庫を補充する。所持上限99個を超えて購入できない。

|---|---|---|---:|---|

| `protective_board` | 防護板 | 工作台建設 | 2 | 標準。設置物の基本 |
| `smoke_canister` | 煙幕筒 | `workshop_combat`研究済み | 2 | 高め。射程制御 |
| `iron_stake` | 鉄杭 | `workshop_traps`研究済み | 2 | 標準。移動低下と罠 |
| `flash_canister` | 閃光筒 | `workshop_advanced`と耐熱加工記録 | 1 | 高め。よろめき付与 |
| `return_flare` | 帰還信号弾 | `workshop_advanced`と信号技術資料 | 1 | 高め。成果確定の保険 |

研究後販売品は、施設で自作する選択肢も残す。商店価格は、素材から製作するより便利だが割高にする。

### 基礎素材

基礎素材は不足補助として少量だけ販売する。通常の素材集めは遠征と再調査で行う。

| ID | 日本語名 | 入荷条件 | 在庫 | 用途 |
|---|---|---|---:|---|
| `wood` | 木材 | 開拓拠点 | 3 | 初期施設、工作品 |
| `herb` | 薬草 | 薬草群生地を安全帰還 | 3 | 治療品、診療所 |
| `hide` | 獣皮 | 灰枝の森攻略 | 2 | 訓練場、探索道具、特性 |
| `iron` | 鉄材 | 沈黙した監視所群攻略 | 2 | 鍛冶、工房、設置物 |
| `cloth` | 織物 | 旧辺境集落到達 | 2 | 医療品、信号具 |
| `food` | 食料 | 旧辺境集落到達 | 3 | 野営食、補給 |

基礎素材の価格は、その素材を使う代表的な消耗品を直接買うより少し割高にする。素材購入だけで
施設研究を進めることは可能だが、在庫が少ないため遠征より効率は悪い。

### 情報商品

情報商品は消耗品枠を使わず、購入時点で次回遠征準備画面へ表示を追加する。

| ID | 日本語名 | 入荷条件 | 効果 |
|---|---|---|---|
| `shop_enemy_briefing` | 敵情勢メモ | `command_scout_network` | 選択地域の次地点の敵カテゴリを表示 |
| `shop_terrain_briefing` | 地形メモ | `command_map_room` | 選択地域の次地点の地形傾向を表示 |
| `shop_hazard_notice` | 危険地点メモ | `command_expedition_planning` | 公開済み危険地形、増援、Boss予告の一部を表示 |

情報商品は未発見の隠し副目標、未加入人物、正確なランダム配置、Bossの詳細HPを表示しない。
司令所研究より強い情報を商店単体で売らない。

### 便利品

| ID | 日本語名 | 入荷条件 | 効果 |
|---|---|---|---|
| `prepared_supply_bundle` | 遠征補給包 | 開拓拠点 | 購入後、遠征準備で基本消耗品候補をまとめて表示する。完成品は別途購入 |
| `storage_sorting_ticket` | 倉庫整理券 | 倉庫上限UI実装後 | 受取保留から倉庫へ移す画面を開く。容量は増やさない |

便利品は戦闘能力、バッグ枠、探索道具枠、装備枠、施設枠を増やさない。

## 販売禁止品

次の品目は商店で売らない。

- 武器本体、分岐武器、Boss武器、敵専用武器
- 調整特性の完成品
- 探索道具の完成品
- 高品質薬草、高品質鉄材、灰晶、希少素材などの高品質・希少素材
- 灰角の大牙、灰霧の大牙、崩落核素材、最終キー素材などのキー素材
- 重要発見、地域踏査記録、医療記録、信号技術資料、訓練記録
- 仲間加入、兵種解放、Skill解放、連携作戦解放
- 拠点段階上昇、地域解放、Boss撃破扱い

例外を作る場合は、この文書へ明記し、対応する地域・施設の価値が壊れない理由を記録する。

## 在庫更新

- 在庫は安全帰還、敗北帰還、地域攻略後に更新する
- 同じ拠点画面を開閉しても在庫は復活しない
- 地域攻略で入荷条件を満たした場合、次に拠点へ戻った時点で反映する
- 在庫0の品は非表示にせず、売り切れとして表示する
- 倉庫上限に達している品は購入不可にし、不足している空き数を表示する

## 価格方針

初期実装では具体価格を固定しない。価格を実装する時は次の比率を守る。

| 商品 | 価格方針 |
|---|---|
| 基本消耗品 | 遠征1回の標準報酬で2〜4個買える程度 |
| 状態異常対策・帰還保険 | 基本消耗品の2〜4倍 |
| 基礎素材 | その素材を使う代表的な完成品より割高 |
| 情報商品 | 消耗品1個前後。情報の強さに応じて上げる |
| 便利品 | 安め。戦闘能力を上げないため負担にしない |

商店価格で難易度を調整しすぎない。難易度は敵AI、地域構造、消耗継続、Objectiveで作る。

## 売却

初期実装では売却を採用しない。売却を追加する場合は次の制限を守る。

- 売却可能: 通常素材、消耗品
- 売却不可: 武器、調整特性、探索道具、キー素材、Discovery、加入候補、地域進行
- 売却価格は購入価格より低くする
- 売却確認では品名、数量、再取得可否を表示する
- 装備中、遠征準備中、受取保留中の品は直接売却できない

## UI

商店画面は次のタブに分ける。

| タブ | 内容 |
|---|---|
| 消耗品 | 購入可能な消耗品、所持数、倉庫上限、使用場所 |
| 素材 | 少量販売の基礎素材、在庫、主な用途 |
| 情報 | 敵情勢メモ、地形メモ、危険地点メモ |
| 便利品 | 補給包、整理券など |

各商品は「価格、在庫、所持数、購入後所持数、使用場所、入荷条件、購入不可理由」を表示する。
色だけに依存せず、購入不可理由を文章で出す。

## セーブ

保存対象:

- 所持クラウン
- 商店在庫
- 入荷済み商品ID
- 購入済み情報商品の次回遠征反映状態
- 価格表Version

商店在庫はロード時に入荷条件から再計算してよいが、同じ拠点滞在中に購入した在庫減少を復元できる
必要がある。価格表Versionが変わっても、既に所持している品は変換しない。

## 受入条件

1. 基本消耗品は商店解放後に購入できる。
2. 研究前の万能薬、煙幕筒、帰還信号弾などは購入できない。
3. 武器、分岐武器、高品質素材、キー素材、Discoveryを購入できない。
4. 在庫0の商品は購入できず、売り切れ表示になる。
5. 倉庫上限99または999を超えて購入できない。
6. 購入失敗時にクラウンだけ減らない。
7. 商店だけで拠点段階上昇、地域解放、兵種解放が完結しない。
8. 情報商品は隠し副目標、正確な乱数配置、未発見人物を表示しない。
9. セーブ後ロードしても所持クラウン、在庫、購入済み情報が復元される。
