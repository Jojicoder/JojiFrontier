# 所持品・素材一覧

文書種別: **説明文書**
参照正本: [`item_system.md`](item_system.md)、[`class_reference.md`](class_reference.md)、
[`character_progression.md`](character_progression.md)、[`base_development.md`](base_development.md)、
[`stage_facility_progression.md`](stage_facility_progression.md)、[`inventory_overflow.md`](inventory_overflow.md)、
[`shop_system.md`](shop_system.md)

この文書は、JOJIFrontierで扱う所持品を分類して確認するための一覧である。効果、数値、
製作費、解放条件、上限、保存処理は各正本を優先する。共有世界の経済方針は
`../JojiWorldBible/docs/world/economy.md`、SRPG消耗品方針は
`../JojiWorldBible/docs/story/war_of_rivermark/game_systems_ja.md`を参照する。

## 大分類

| 分類 | 所有場所 | 遠征中の扱い | 主な用途 |
|---|---|---|---|
| 消耗品 | 倉庫、遠征バッグ | 6枠へ持ち込み、使うと減る | 回復、帰還、設置物、戦闘補助 |
| 探索道具 | 倉庫、探索道具枠 | 2枠へ装備、通常使用では減らない | 探索選択肢、情報、消耗軽減 |
| 武器 | 共有倉庫、ユニット装備 | 1人1本、遠征中は変更不可 | 攻撃威力、射程、兵種役割の調整 |
| 調整特性 | 共有倉庫、武器装着 | 1部品を1武器へ装着 | 地形・状態・ノックバック対策 |
| 通常素材 | 倉庫、Pending Loot | 安全帰還で確定、製作や研究に消費 | 施設、消耗品、武器、道具 |
| 高品質素材 | 倉庫、Pending Loot | 通常素材より取得機会が少ない | 上位製作、専門研究 |
| キー素材 | 倉庫、Pending Loot | 進行用途限定。通常製作では使わない | 拠点段階上昇、地域攻略証明 |
| 重要発見 | Discovery Registry | 数量なし、消費不可 | 施設研究、地域解放、会話解放 |

## 消耗品

遠征バッグに6個まで持ち込む。同じ消耗品を複数持ち込める。未使用品は帰還・敗北時に倉庫へ戻り、
使用済み品は戻らない。

| ID | 日本語名 | 使用場所 | 役割 |
|---|---|---|---|
| `first_aid_kit` | 救急セット | キャンプ | 味方1人のHPを大きく回復 |
| `field_treatment_kit` | 野戦治療キット | 戦闘 / キャンプ | 味方1人のHPを小回復 |
| `rescue_pack` | 救命包 | キャンプ | 戦闘不能者を復帰 |
| `camp_rations` | 野営食 | キャンプ | 生存者全体の小回復と次地点の事前消耗軽減 |
| `panacea` | 万能薬 | 戦闘 / キャンプ | 状態異常を解除 |
| `protective_board` | 防護板 | 戦闘 | 隣接マスに障害物を設置 |
| `smoke_canister` | 煙幕筒 | 戦闘 | 射程を制限する煙幕マスを作る |
| `iron_stake` | 鉄杭 | 戦闘 | 罠を設置し、敵へ固定ダメージと移動低下 |
| `flash_canister` | 閃光筒 | 戦闘 | 敵へよろめきを付与 |
| `return_flare` | 帰還信号弾 | 地点間 / キャンプ | 次の危険地点へ進まず安全帰還 |
| `protection_crate` | 保護箱 | 地点間 / キャンプ | 通常戦利品1スタックを敗北ロストから保護 |
| `scouting_chart` | 偵察図 | 遠征開始時 | 次地点の敵種・地形傾向・公開Objectiveを確認 |

現行`ItemType`へ実装済みなのは`first_aid_kit`/`field_treatment_kit`/`rescue_pack`/
`camp_rations`/`protective_board`/`return_flare`の6種のみ。`panacea`/`smoke_canister`/
`iron_stake`/`flash_canister`/`protection_crate`/`scouting_chart`は正本記載の設計済み項目で、
コード・UIには未接続。

## 探索道具

工房で一度製作すると恒久取得する。遠征ごとに2種類を選ぶ。探索道具は兵種専用選択肢の完全代替
ではなく、対応兵種がいる場合の方が情報、配置、報酬のいずれかで優れる。

| ID | 日本語名 | 主な用途 |
|---|---|---|
| `climbing_gear` | 登攀具 | 崖、高所、棚道の調査 |
| `mining_tools` | 採掘具 | 鉱石採取、軽い崩落除去 |
| `dismantling_tools` | 解体具 | 機械、罠、封鎖装置の調査 |
| `lighting_gear` | 照明具 | 洞窟、地下、暗い遺跡の情報公開 |
| `portable_rope_bridge` | 携帯縄橋 | 川、亀裂、谷の横断 |
| `hunting_gear` | 狩猟具 | 足跡追跡、食料・獣素材の採取 |

## 実装上の注意

- UIでは素材、武器、消耗品、探索道具、重要発見を同じ「アイテム」として混ぜて表示しない
- 倉庫画面は「消耗品」「装備」「素材」「重要品」のタブに分ける
- 戦闘中の行動メニューには、使用可能な消耗品だけを「アイテム」一覧として表示する
- 遠征準備画面では「消耗品6枠」と「探索道具2枠」を分ける
- ユニットページでは武器、調整特性、装備スキルを扱い、素材や消耗品は扱わない
- 施設ページでは製作・研究・必要素材を表示し、装備変更そのものはユニットページへ送る
- 重要発見とキー素材は不足条件として表示するが、通常の素材消費リストへ混ぜない
- 商店画面では購入可能な消耗品、少量の基礎素材、情報商品、便利品だけを扱い、武器・高品質素材・
  キー素材・重要発見は売らない
