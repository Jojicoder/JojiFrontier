# JOJIFrontier 拠点発展システム

状態: JOJIFrontier固有のゲームシステム案。地域名や世界設定へ接続する際は
`../JojiWorldBible` を確認して正史との整合を取る。

## 基本方針

拠点は施設レベル制ではなく、開拓段階と分岐ノードで発展する。重要地点の
確保は拠点全体の発展上限を開くが、個々の技術を自動解放はしない。

基本循環:

1. 遠征で知識、人物、施設、素材を発見する。
2. 無事に帰還し、未確定の発見物と戦利品を恒久資産にする。
3. 前提発見と素材を揃えて施設ノードを解放する。
4. 新しい戦闘、探索、編成、装備の選択肢を次の遠征で使う。

帰還前の発見物と素材は敗北時に失う。解放済みノードは恒久で失われない。

## 拠点全体の開拓段階

| 段階 | 名称 | 主な範囲 |
|---|---|---|
| 0 | 野営地 | 初期4人編成、基本物資 |
| 1 | 開拓拠点 | 各施設の第1分岐、地域情報 |
| 2 | 辺境集落 | 専門兵種、専門装備、探索道具 |
| 3 | 開拓都市 | 上位分岐、深層遠征、特殊遠征 |

段階はノードへのアクセス上限であり、技術そのものではない。

### 第1段階の施設枠

作戦テント、倉庫、共同テントはゲームの基本ループに必要な基礎設備であり、施設枠を
消費せず常時利用できる。

開拓拠点の有効施設枠は2。救護テント、訓練場、工作台、簡易鍛冶台から選んで建設する。
施設は解体可能で、各使用素材の50%を端数切り捨てで返却する。解体してもDiscovery、
解放技術、製作済みアイテム、武器は失われないが、その施設を必要とする操作は再建まで
利用できない。次の拠点段階で有効枠を増やす。

## 共通の解放条件

全ノードの条件は次の分類で統一する。

- 必要開拓段階
- 持ち帰った発見、記録、設計図
- キー素材と一般素材
- 前提施設ノード
- 必要に応じて専門家の加入

施設効果は原則として新しい行動、ルート、アイテム、編成判断を最低1つ追加する。
単純な数値上昇は副次効果に留める。

## 1. 司令所

初期ノード「作戦テント」:

- 遠征先を選択
- 4人編成を確認
- 基本的な危険度を表示

分岐:

- 偵察網: 敵の種類、敵の初期配置を表示
- 地図室: ルート分岐の一部、ボスまでのおおよその距離を表示
- 遠征計画室: 深層情報を表示し、特殊遠征を解放

## 2. 訓練所

初期ノード「訓練場」: 基礎兵種と基本兵種技を使用可能にする。

分岐:

- 前衛訓練: 古参兵、槍兵、重装兵技術
- 機動訓練: 斥候、猟兵、伝令騎兵技術
- 専門訓練: 工兵、旗手、上位兵種分岐

兵種解放は通常レベル成長ではなく、盤面能力と探索選択肢を増やす。

## 3. 鍛冶場

初期ノード「簡易鍛冶台」:

- 武器の装備変更
- 武器ごとに調整特性を1つ装着
- 調整変更は遠征前のみ可能
- 武器分岐レシピの閲覧と製作
- Iron系の基本武器を使用可能

通常の使用回数制武器耐久は採用しない。武器は通常使用で破損せず、遠征中の修理操作も
行わない。鍛冶場は性能を元へ戻す施設ではなく、戦い方を変更する施設とする。

分岐:

- 武器鍛造: 高品質武器、武器固有特性
- 重装加工: 防具強化、ノックバック耐性装備
- 特殊素材加工: 魔物素材武器、遺跡素材武器

武器は `+1` から `+10` ではなく機能別に分岐する。例としてIron Spearは、
射程型Long Spear、ノックバック型Heavy Spear、迎撃型Guard Spearへ分かれる。

成長順序:

```text
基本武器
-> 小規模調整（特性1枠）
-> 新素材・Discovery・鍛冶技術を獲得
-> 武器分岐
-> 数値上位ではなく戦い方が変化
```

最初の調整特性:

| 特性 | 費用 | 効果 |
|---|---|---|
| 獣皮の柄巻き | 獣皮1 | 戦闘ごとに最初に受けるノックバックを1回無効化 |

最初は獣皮の柄巻きだけを使用可能にし、調整システムを一度に増やしすぎない。

将来のIron Spear分岐:

- Long Spear: 射程型。最大射程+1、威力-2
- Heavy Spear: 火力型。威力+2、MOV-1
- Guard Spear: 迎撃型。Brace強化、通常威力-1

灰角の大牙は開拓拠点への発展に使うキー素材であり、武器製作には使わない。将来、灰角系の
武器調整を追加する場合は別素材「灰角の欠片」を使用し、必須進行素材と競合させない。

## 4. 診療所

初期ノード「救護テント」: 応急手当具と救命包を解放する。

分岐:

- 野戦医療: 野戦治療具、高度治療具
- 救命技術: 高級救命包、復帰時HP増加
- 薬学: 万能薬、特殊状態異常治療

拠点帰還時は常に完全回復する。診療所は拠点回復ではなく、遠征中の医療能力を
強化する施設とする。

## 5. 工房

初期ノード「工作台」: 基本探索道具を解放する。

分岐:

- 探索工作: 登攀具、採掘具、照明具
- 戦闘工作: 防護板、鉄杭、煙幕筒
- 高度工作: 帰還信号筒、保護箱、特殊遺跡装置

## 6. 宿舎

初期ノード「共同テント」: 初期仲間4人と4人編成を使用可能にする。

分岐:

- 宿舎増築: 新しい仲間の受け入れ枠
- 専門区画: 特殊兵種の仲間を受け入れ可能にする
- 交流区画: 仲間イベント、連携能力

施設解放は受け入れを可能にするだけで、仲間は探索や地域イベントを通じて加入する。
同一兵種の出撃制限は未確定とする。厳格な禁止より「同一兵種2人まで」または追加の
補給コストを優先候補とし、編成テスト後に決定する。

## 最初の縦切り実装

現在の3戦遠征との暫定接続:

| 遠征成果 | 安全帰還後に得る発展機会 |
|---|---|
| Cinderwatch Gate確保 | 司令所「偵察網」の発見条件 |
| Ironwatch Storesの記録回収 | 診療所「野戦医療」の発見条件 |
| Signal Tower復旧 | 工房「帰還信号技術」の発見条件 |
| 地域キー素材を持ち帰る | 開拓段階1への発展資格 |

これらの地名と歴史的背景は暫定的なFrontierシナリオであり、確定時にWorld Bibleを
検証する。

## 実装時に永続化するデータ

- 現在の開拓段階
- 帰還済みの発見ID
- 恒久素材在庫
- 解放済み施設ノードID
- ノードから導出される使用可能兵種、アイテム、武器、遠征情報

解放条件はUIにハードコードせずデータから評価する。既存の遠征バッグ、未確定戦利品、
帰還時確定、敗北時喪失のルールは変更しない。
