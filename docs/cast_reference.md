# JOJIFrontier 遠征隊人物設定

更新日: 2026-07-13

## 正史上の扱い

12人の兵種、加入順、Frontier内での役割と人物設定はJOJIFrontierの正式設定とする。
名前、出身地の細部、経歴はWorld Bibleへの登録待ちであり、共有正史上はFrontier提案として扱う。

参照したWorld Bible:

- `../../JojiWorldBible/docs/world/embermarch.md`
- `../../JojiWorldBible/military.md`
- `../../JojiWorldBible/naming.md`
- `../../JojiWorldBible/characters.md`
- `../../JojiWorldBible/culture.md`
- `../../JojiWorldBible/religion.md`

既存のRhaest Bloodmarch、Dael Bloodmarch、Ser Idris Ironwatch、Soren Scorchreachは王家、軍上層、
著名な斥候であり、本作の通常遠征隊員として流用しない。新しい姓はエンバーマーチの軍事、火、
前線、地点由来の命名傾向に合わせた提案である。

## 初期6人

### ライダン・アッシュウォッチ

| 項目 | 設定 |
|---|---|
| 兵種 | 行軍隊長 |
| 性別 | 男性 |
| 外見 | 灰黒の短髪。左眉に古い裂傷。軽い指揮外套と使い込んだ剣 |
| 前歴 | 前線防衛線の小隊副官。大部隊を動かす地位ではなく、孤立した分遣隊の判断を任されてきた |
| 参加理由 | 本隊を弱めずに失われた街道と監視所を取り戻す任務を引き受けた |
| 個人的目的 | 「帰還できる道」を地図に残し、次の隊を同じ場所で失わない |
| 長所 | 判断を保留せず、撤退も命令として言える |
| 欠点 | 責任を自分だけで背負い、仲間へ理由を説明するのが遅い |
| 続行判断 | 情報と退路があれば続行。成果だけを理由に無理をしない |
| 口調 | 短い命令形だが威圧しない。「決める」「戻す」「残す」を多用 |
| 禁止 | 万能な名将、王家の密命を知る人物、常に正しい主人公にしない |

主な関係:

- マエラを現場判断の基準として信頼するが、慎重論に従うだけではない
- レッサとは指揮と士気の責任範囲を巡って意見が分かれる
- ヴェイラから、街道確保が獣と人の衝突を増やす危険を指摘される

会話軸: 指揮責任、帰還判断、開拓の範囲、元守備隊長との対比。

## 口調共通規則

- 日本語表示を正規とする
- 現代的なネット用語、軽すぎる掛け合い、メタ発言を使わない
- 長い世界説明を1人へ集中させない
- 道、見張り、食料、鋼、天候、負傷、持ち帰れる量から話を始める
- 同じ情報を複数人が説明する場合は、事実を増やさず見解だけを変える
- March Line外側の正体を知っているように話させない
- 既存Canon人物との親族・直属師弟関係をBible承認なしで追加しない
