# 地点報酬と施設強化の対応表

状態: JOJIFrontier固有の進行仕様として確定。全62地点の戦闘・探索成果を施設発展へ接続する正本。
地点の敵、Objective、正確な報酬個数は`regions/`以下、施設自体の効果は
[`base_development.md`](base_development.md)を正本とする。
研究ノードの完全な条件と効果は[`facility_research.md`](facility_research.md)を正本とする。

## 解放フロー

1. 地点のObjectiveを達成し、通常素材を`PendingLoot`、記録・知識を`PendingDiscoveries`へ入れる。
2. 敗北または撤退失敗では未確定成果を失う。
3. 安全帰還時にStorageとDiscovery Registryへ確定する。
4. Discoveryは施設ノードを「研究可能」にするだけで、自動建設・自動強化しない。
5. 拠点画面で必要な通常素材を支払い、プレイヤーが強化を確定する。

重要発見は参照条件であり消費しない。ボスキー素材は通常の施設研究・武器・道具製作では消費せず、
`base_development.md`の拠点段階上昇表で明示された場合だけ、その段階上昇専用素材として消費する。
灰角の大牙は野営地から開拓拠点への発展でのみ消費できる。通常の施設建設・研究費は通常素材だけを
消費する。
同じDiscoveryの再取得、同じObjectiveイベントの再処理、周回による再攻略で解放を二重付与しない。

## UI表示

施設画面では各ノードに「必要な発見」「必要素材」「取得地域・地点」「効果」を日本語で表示する。
未到達地点の固有名は伏せて「未確認地域の記録」とし、取得済みDiscoveryだけ実名表示する。
不足条件は赤、充足条件は白、研究可能は黄、解放済みは緑で区別し、色だけに依存せず文言も併記する。
