# JOJIFrontier 全体あらすじ

## 正典上の位置づけ

この文書はJOJIFrontierのキャンペーン用物語案である。エンバーマーチ、前線防衛線、
暁の教会、南方辺境に関する共有設定はWorld Bibleを正本とする。

参照した正本:

- `../../JojiWorldBible/docs/world/embermarch.md`
- `../../JojiWorldBible/locations.md`
- `../../JojiWorldBible/docs/world/geography.md`
- `../../JojiWorldBible/religion.md`
- `../../JojiWorldBible/docs/world/flora_fauna.md`

World Bibleでは前線防衛線の外側に何があるかを意図的に確定していない。本作も、その正体、
文明、勢力、世界の終端を断定しない。灰枝の森以降の地域名、失われた記録、登場人物、個別事件は
Frontier側の提案であり、共有正典へ登録されるまでは他作品の確定事実にしない。

## 一文概要

4人の遠征隊を率いて失われた街道と監視所を調査し、成果を生きて持ち帰りながら、仮設野営地を
人が帰ってこられる開拓都市へ育てる物語。

## 導入

エンバーマーチ南部の前線防衛線。その外側に置かれていた小規模な監視所群が、大規模な灰嵐の
夜を境に沈黙した。

前線防衛線の本隊は国境を維持するだけで精一杯であり、大部隊を調査へ送る余裕がない。
プレイヤーは仮設開拓隊の**遠征隊長**に任命され、少数の仲間、6枠の遠征荷物、粗末な野営地を
与えられる。

任務は外側を征服することではない。

- 失われた街道を取り戻す
- 監視所と信号網を再接続する
- 木材、薬草、鉄材、道具、記録を持ち帰る
- 遠征隊と開拓民が帰還できる拠点を築く
- 監視所群が沈黙した理由を調べる

遠征中に見つけた戦利品や重要発見は、拠点へ安全帰還するまで正式な所有物にならない。
奥へ進むほど成果は増えるが、敗北すればその遠征の未確定資産を失う。

## エンバーマーチとの接続

### プレイヤーの立場

プレイヤーはエンバーマーチの王、戦王、前線防衛線総司令官ではない。前線防衛線の外縁調査を
任された現地の**遠征隊長**である。

- 任命元は前線防衛線の地域司令部
- 指揮できるのは開拓拠点、遠征隊、現地で確保した施設
- 国家全軍、外交、徴税、王位継承を決定する権限は持たない
- 4人編成、遠征先、持ち込み物資、施設運用、帰還判断を担当する
- 統治家ブラッドマーチ家と前線司令部は背景に存在するが、序盤から直接指示を出し続けない

これにより、エンバーマーチ全体を動かす戦略ゲームではなく、国家の大きな防衛線を支える
小規模開拓遠征として物語を成立させる。

### なぜ4人で遠征するのか

エンバーマーチは前線防衛線の維持を最優先しており、大部隊を外縁調査へ回せない。少人数編成は
単なるゲーム上の都合ではなく、国家の慢性的な人員不足と防衛優先の軍事思想から生じる。

- 主力軍は前線防衛線を離れられない
- 大部隊は補給量が増え、崩れた街道を進みにくい
- 4人なら偵察、護衛、治療、戦闘の役割を組み替えられる
- 失敗しても前線全体を弱体化させない
- 成果が確認できた地域だけ、後から開拓民と補給隊を送る

### シンダーウォッチと開拓拠点

シンダーウォッチは独立国家や新しい領地ではない。エンバーマーチの前線防衛線を補助するため、
外縁に再設置された小規模な中継拠点である。

```text
エンバーマーチ内地
-> 前線防衛線
-> シンダーウォッチ開拓拠点
-> 灰枝の森
-> 沈黙した監視所群
-> 地図外縁
```

拠点が発展しても国家から独立しない。野営地、開拓拠点、辺境集落、開拓都市という段階は、
現地の補給能力、受入人数、施設機能が増えることを表す。

### 拠点施設との接続

| 施設 | エンバーマーチでの意味 |
|---|---|
| 司令所 | 前線司令部から届く命令、地図、監視所情報を遠征向けに整理する |
| 訓練所 | 実戦経験を兵種技術として共有し、4人編成の戦術を増やす |
| 鍛冶場 | 輸入鉄材と現地素材を使い、基本武器を遠征向けに調整する |
| 診療所 | 暁の教会の野戦医療を、遠征中の治療手段へ転用する |
| 工房 | 橋、障害物、信号装置、罠、探索道具を整備する |
| 宿舎 | 前線本隊から回せる専門家と現地加入者を受け入れる |
| 倉庫 | 国家支給品、持ち帰った通常素材、確定済み戦利品を分けて管理する |

施設の成長は個人レベルの代わりであり、エンバーマーチの軍事国家としての蓄積を現地へ少しずつ
移す過程でもある。

### 兵種との接続

初期兵種と序盤加入兵種は、エンバーマーチの防衛文化から生まれた役割を中心にする。

- 槍兵、古参守備兵、監視弓兵、辺境斥候、重装兵は前線防衛の典型的役割
- 行軍隊長は4人遠征の現場指揮を担当する
- 暁の衛生兵は、エンバーマーチで重視される暁の教会の野戦医療を担う
- 辺境工兵、伝令騎兵、旗手は失われた監視網の再建に必要な専門家
- 辺境猟兵は野生動物を単に討伐せず、生態、罠、進路を理解する
- 戦闘魔導士は一般兵ではなく、特定の調査目的で参加する希少な名前付き人物

兵種は国家軍の階級一覧ではなく、JOJIFrontierの3×8盤面と探索選択肢へ落とし込んだ
ゲーム上の役割である。

### 戦利品と国家経済

エンバーマーチは前線維持のため鉄や鋼などの軍需資源を必要とする。本作で木材、薬草、鉄材、
石材、旧軍備を持ち帰る行為は、単なる製作素材集めではなく、補給に乏しい外縁拠点を自立させ、
前線本隊の負担を減らす行動になる。

- 未確定戦利品は遠征隊が持ち運んでいる状態
- 安全帰還後に検品され、拠点倉庫の正式資産になる
- 重要発見は売却物ではなく、前線技術や施設解放へ使う記録になる
- 地域ボス素材は地域を安全化した証明と、拠点発展の実用品になる
- 拠点資産は遠征隊の私有財産ではなく、シンダーウォッチの運用資産として扱う

### 物語上の国家への影響

開拓成功はエンバーマーチ全体の戦争を終わらせない。効果は具体的で地域的なものに限定する。

- 失われた補給路が一本戻る
- 前線防衛線の監視範囲が広がる
- 木材、薬草、鉄材の供給源が増える
- 孤立した人員と記録を回収できる
- 主力軍を動かさずに外縁情報を得られる

最終的に開拓都市が成立しても、エンバーマーチの前線問題がすべて解決するわけではない。
JOJIFrontierの成果は「前線の外側へ継続して調査できる足場を作ったこと」とする。

### 共有設定を守る境界

- 前線防衛線の外側にある脅威の正体を確定しない
- エンバーマーチの統治家、外交関係、歴史をFrontier側だけで変更しない
- 戦王や既存の著名人物を、通常の4人遠征隊員として扱わない
- シンダーウォッチを独立国家やブラッドマーチ家に対抗する領地にしない
- 魔法を一般化せず、学習可能だが希少というWorld Bibleの規模を維持する
- 野生動物をすべて魔物や国家的な敵にしない
- 新しい地域名、人物、過去事件は共有正典へ登録されるまでFrontier提案として扱う

## 第一部 道を開く

最初に立ちはだかるのは、悪意を持つ軍勢ではない。街道予定地を縄張りとする野生動物と、
長期間放置された森そのものである。

遠征隊は灰枝の森を踏査し、薬草の沢を調べ、街道を塞ぐ灰角大猪を森の奥へ退かせる。
大猪は魔物ではなく、人間の進出によって縄張りを脅かされた大型獣として描く。

```text
仮設野営地
-> 灰枝の森を踏査
-> 街道を確保
-> 木材と薬草の供給開始
-> 開拓拠点を建設
```

ここでプレイヤーは、開拓とは敵を倒すことではなく、安全な通行路を作り、得た成果を
持ち帰ることだと学ぶ。

## 第二部 沈黙した監視所群

森の先には、灰嵐の後に放棄された監視所、物資庫、兵舎、信号塔が残されている。

施設には新しい足跡があり、武器と医療品が持ち出され、信号装置は人為的に停止されている。
元守備兵、脱走兵、回収団は、それぞれ異なる事情で旧施設を利用している。

調査を進めると、元守備隊長が信号網を意図的に遮断したことが判明する。彼は単なる略奪者では
ない。救援が来ないと判断し、生存者と物資を一か所へ集め、外側へ人間の位置を知らせないために
信号を止めていた。

遠征隊は「街道と信号を戻して開拓を再開する」という立場から彼と対立する。旧守備隊を退けて
信号塔を復旧すると、地図に記されていない遠方から一度だけ応答が返り、すぐに途絶える。

この地域の複数回の遠征と安全帰還を通じて、辺境工兵、伝令騎兵、旗手が仲間になる。

## 第三部 拠点を存続させる

遠方の応答を追うには、一度の遠征では足りない。長期的に人員と物資を送り出せる拠点が必要に
なる。

遠征隊は灰鉄採石場へ向かい、鉄鉱石と建築用石材の供給路を確保する。採掘記録の中から、通常の
鉱脈とは異なる反応を記した古い調査資料が見つかり、その調査に関わる希少な戦闘魔導士が遠征隊へ
加わる。

灰鉄採石場から安全帰還した時点で、全12兵種を編成できるようになる。以降の成長は兵種そのもの
ではなく、武器分岐、追加スキル、調整特性、施設技術によって戦い方を広げる。

```text
開拓拠点
-> 採石場を確保
-> 建築材と鉄材を供給
-> 全12兵種が合流
-> 辺境集落へ発展
```

## 第四部 開拓する側の責任

黒水低湿地、風裂き高原、旧辺境集落では、自然環境だけでなく、すでにその土地を利用している
人々や別の回収集団と向き合う。

- 毒と病が残る湿地で、薬草を採るか安全を優先する
- 崩れた高原街道で、物資と負傷者のどちらを先に運ぶか選ぶ
- 旧集落を利用する人々と、交渉、護衛、戦闘のいずれで関係を作るか判断する
- 開拓拠点の限られた資源を、どの施設・研究・製作へ使うか決める

ここでは敵全滅だけでなく、防衛、護衛、脱出、地点確保、装置操作が主要な任務になる。
遠征隊の目的は土地からすべてを排除することではなく、継続して使える道と合意を作ることである。

## 第五部 失われた記録

燼火峡谷と埋没聖堂の調査によって、灰嵐以前にも外縁の監視所が何度か沈黙していたことを示す
断片的な記録が見つかる。

暁の教会が残した医療記録には、多数の負傷者、避難、物資移送の記述がある。しかし記録は欠けて
おり、原因が獣害、自然災害、人間同士の争い、それ以外の何であったかは断定できない。

魔法は調査手段の一つにすぎず、過去や外側の正体を自動的に説明する答えにはしない。

## 第六部 切り離された防衛線

破砕された前線砦には、外縁の監視所群を切り離した命令の断片が残されている。

命令を出した者は、明確な裏切者とは限らない。限られた兵力を前線防衛線へ戻し、内側を守るため、
外縁の信号と補給を断とうとした可能性がある。しかし撤退命令は最後まで届かず、複数の監視所と
人員が孤立した。

遠征隊は砦を確保し、最後に残された不完全な座標を得る。その先は、既存の地図が終わる場所で
ある。

## 最終部 地図外縁

地図外縁は9地点からなる本編最長の遠征である。これまでに解放した兵種、施設、アイテム、
探索能力を組み合わせなければ、安全な帰路を作れない。

最終目標は、外側にいる何かを完全に倒して世界の謎を解くことではない。

- 最後の監視記録を回収する
- 生存者または人の活動痕跡を探す
- 新しい監視地点を設置する
- 拠点まで戻れる街道を確保する

遠征隊は、地図の先にも調査すべき土地と危険が続いていることだけを確認する。外側の全容は
断定せず、World Bibleの意図的な空白を維持する。

安全帰還によって開拓拠点は都市段階へ到達し、前線防衛線の外側へ継続的な遠征を行えるように
なる。本編終了後は、さらに深い地域へ進む任意遠征を解放できる。

## 物語の中心

JOJIFrontierは、ひとりの英雄が世界を救う物語ではない。

```text
4人を選ぶ
-> 遠征する
-> 判断する
-> 戦う
-> 生きて帰る
-> 成果で拠点を育てる
-> 次はさらに遠くへ進む
```

プレイヤーが最終的に作るものは最強の個人ではなく、**人が帰ってこられる場所と、その先へ
続く道**である。

## 表記方針

- ゲーム画面の地域名、施設名、兵種名、アイテム名、任務名は日本語を主表示とする
- 英字の地域ID、ユニットID、重要発見IDは内部処理とセーブデータだけに使用する
- 内部IDを通常HUD、戦闘結果、施設画面へ表示しない
- World Bibleの英語固有名を日本語表示する場合、意味を変える意訳を避け、表記対照表で一元管理する
- 現時点では`Embermarch`を「エンバーマーチ」、`The March Line`を「前線防衛線」、
  `Church of Dawn`を「暁の教会」、`Cinderwatch`を「シンダーウォッチ」と表示する
- これらの日本語表記はFrontierの表示規約であり、World Bibleの英語正本を改名しない
